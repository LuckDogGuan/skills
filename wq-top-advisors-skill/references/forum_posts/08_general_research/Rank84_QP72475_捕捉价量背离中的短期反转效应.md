# 捕捉价量背离中的短期反转效应

- **作者**: 顾问 QP72475 (点塔王) (Rank 84)

## 帖子正文

```
-1 * correlation(rank(high), rank(volume), 5)
```

让我们逐层拆解这个公式：

1. **`rank(high)`** : 对所有股票在当天的 **最高价** 进行横截面排名。这意味着，在每一个交易日，我们计算所有股票的最高价，然后根据这个价格从低到高进行排序（通常是1为最低，N为最高），并将价格转换为排名序号。
2. **`rank(volume)`** : 对当天的 **成交量** 进行同样的横截面排名。
3. **`correlation(..., ..., 5)`** : 计算上述两个排名序列在过去 **5个交易日** 内的相关系数。这是一个时间序列相关性，衡量的是在过去一周里，一只股票的“最高价排名”和“成交量排名”之间的线性关系。
4. **`-1 * ...`** : 最后，将计算出的相关系数乘以  **-1** ，即取其相反数。

### **核心逻辑与金融直觉解析**

这个因子的核心思想是捕捉 **价量背离** 中的短期反转效应。

#### **1. 正相关 vs. 负相关**

- **正相关** ： 如果一只股票在过去5天里，最高价的排名和成交量的排名是同向变动的（即价升量增，价跌量缩），那么它们的相关系数会是一个 **正数** 。
- **负相关** ： 如果一只股票在过去5天里，出现了“价升量缩”或“价跌量增”的背离现象，那么它们的相关系数会是一个 **负数** 。

#### **2. 因子取反的意义**

由于因子最后乘以了  `-1` ，所以：

- **当相关系数为正时（价量同步）**  -> 因子值为 **负** 。
- **当相关系数为负时（价量背离）**  -> 因子值为 **正** 。

**金融直觉：**

- **我们期望因子值为正的股票未来表现更好。**  根据公式，因子值为正，意味着该股票近期出现了“价量背离”。在经典的量价理论中，“价跌量增”可能预示着卖压的集中释放，但结合“最高价”这个特定价格和短期反转策略的逻辑，更常见的解释是：
  - 当价格（最高价）排名相对较低（下跌），但成交量排名却很高（放量）时，形成了“放量下跌”的格局。这种异常的放量可能意味着恐慌性抛售或主力出货的结束，短期内存在超跌反弹的可能。
  - 另一种背离是“价升量缩”，即最高价排名上升（创新高），但成交量排名下降（缩量），这表明上涨动力不足，可能即将回调。但在这个因子中，由于我们关注的是最高价，可能更侧重于前一种情况。
- **我们期望因子值为负的股票未来表现更差。**  因子值为负，意味着“价量同步”，比如“价升量增”。这在传统技术分析中是健康上涨的信号。然而，在Alpha101的短周期（5天）反转框架下，这可能被解读为短期动能已经充分甚至过度释放，后续买入力量衰竭，价格可能会暂时回落。

**总结一下核心逻辑：** 
 **试图找出那些近期出现“价量背离”（特别是放量下跌或缩量上涨）的股票，并做多它们，同时做空那些“价量同步”（健康上涨）的股票，赌的是一个短期的价格反转。**  这是一个典型的 **短期均值回复** 因子。
