# 结合量价关系与价格位置的多维度信号

- **链接**: 结合量价关系与价格位置的多维度信号.md
- **作者**: 顾问 QP72475 (点塔王) (Rank 84)

## 帖子正文

```
scale(((ts_corr(adv20, low, 5) + ((high + low) / 2)) - close))
```

**逐步拆解：**

- **adv20** ：过去20日的平均成交量
- **ts_corr(adv20, low, 5)** ：计算过去5日内 `adv20` 与 `low` 价格的相关系数
- **((high + low) / 2)** ：当日中点价格（(最高价+最低价)/2）
- **整体结构** ： `相关系数 + 中点价格 - 收盘价`
- **scale()** ：横截面标准化（去均值，除以标准差）

### 2.  **逻辑与经济含义**

- **核心思想** ：捕捉 **成交量趋势与价格形态的背离**
  - 当 `adv20` 与 `low` 正相关时：成交量放大伴随低点抬升 → 可能看涨
  - 加入 `(high+low)/2 - close` ：衡量当日价格相对中点的偏移
- **因子值高可能预示** ：
  - 成交量趋势与低点正相关 + 收盘价低于当日中点 → 潜在反转信号
- **本质** ：结合量价关系与价格位置的多维度信号
- ### **交易逻辑与预期收益**
  - **多头策略** ：选择因子值最高的股票
  - 预期：成交量与低点正相关性强的股票，且收盘价相对中点偏低 → 后续可能反弹
  - **空头策略** ：选择因子值最低的股票
  - 预期：量价背离或收盘价已相对中点偏高 → 可能回调
