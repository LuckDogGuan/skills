# 一个基于价格位置变化率的短期反转因子

- **链接**: 一个基于价格位置变化率的短期反转因子.md
- **作者**: 顾问 QP72475 (点塔王) (Rank 84)

## 帖子正文

```
-1 * delta((((close - low) - (high - close)) / (high - low)), 9)
```

### 一、公式解析

#### 1. 核心成分拆解

**内部比率计算** ：

text

```
((close - low) - (high - close)) / (high - low)
```

可简化为：

text

```
(2*close - high - low) / (high - low)
```

这就是著名的 **收盘价相对位置指标** ，衡量收盘价在当日价格区间中的相对位置。

- 当收盘价接近当日最高价时，该值趋近于1
- 当收盘价接近当日最低价时，该值趋近于-1
- 当收盘价位于中点时，该值为0

#### 2. 时间序列变换

**delta(·, 9)** ：
计算该指标在9天时间窗口内的变化量，即：

text

```
当前值 - 9天前的值
```

**乘以-1** ：
反转变化方向，使因子值与价格变化呈反向关系。

### 二、金融逻辑解读

#### 1. 基本逻辑

该因子捕捉的是 **价格位置动量的反转效应** ：

- 如果收盘价在价格区间中的相对位置连续9天上升（收盘价越来越接近最高价），因子值为负
- 反之，如果相对位置连续下降，因子值为正

#### 2. 市场行为解释

**过度反应假说** ：

- 当股票连续多日收盘接近当日最高价时，可能存在过度乐观情绪
- 因子认为这种趋势不可持续，预期将出现反转
- 因此做空近期强势股，做多近期弱势股

**支撑阻力逻辑** ：

- 连续收盘接近高点可能耗尽买盘力量
- 收盘位置的变化率可能预示着动能衰竭
