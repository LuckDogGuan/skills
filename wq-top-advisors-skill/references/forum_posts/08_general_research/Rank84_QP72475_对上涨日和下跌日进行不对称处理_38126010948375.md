# 对上涨日和下跌日进行不对称处理

## 帖子正文

```
alpha030 =
    CLOSE - DELAY(CLOSE, 1)
    >= 0 ?
        STD(CLOSE, 20) :
    CLOSE - DELAY(CLOSE, 1)
    < 0 ?
        STD(CLOSE, 20) * (CLOSE - DELAY(CLOSE, 1)) / STD(CLOSE - DELAY(CLOSE, 1), 20) :
    0
```

用更清晰的条件语句表述：

- 如果今日收益（收盘价变动） **大于等于0** ，则因子值 =  `STD(CLOSE, 20)` 。
- 如果今日收益 **小于0** ，则因子值 =  `STD(CLOSE, 20) * (今日收益) / STD(收益序列， 20)` 。
- 否则（理论上不会发生），为0。

### 2. 逻辑解析与分步拆解

这个因子的核心是 **对上涨日和下跌日进行不对称处理** 。

**步骤1：计算基础波动率** 
 `STD(CLOSE, 20)`  是过去20日收盘价的标准差，代表股价的绝对波动水平。

**步骤2：条件分支**

1. **在上涨日（Close >= Delay(Close,1)）** ：
   - 因子值直接等于  `STD(CLOSE, 20)` 。
   - **这意味着，无论涨多少，只要上涨，所有股票在该日的因子值都只与其自身过去20日的波动率正相关。**  波动率大的股票，因子值就大。
2. **在下跌日（Close < Delay(Close,1)）** ：
   - 因子值 =  `STD(CLOSE, 20) * (今日收益) / STD(收益序列， 20)` 。
   - 我们可以将其分解为三部分：
   - `STD(CLOSE, 20)` ：基础波动率。
   - `(今日收益)` ：一个负数。
   - `STD(收益序列， 20)` ：过去20日收益率的标准差，代表收益率的波动性。
   - 所以，在下跌日，因子值 =  `基础波动率 * (标准化后的负收益)` 。
   - **直观理解** ：下跌得越多（负收益绝对值越大），因子值 **越小（因为乘以了一个更大的负数）** 。同时，这个下跌幅度还用自身收益波动率进行了标准化，使得不同波动特性的股票之间具有可比性。

### 3. 经济直觉与因子意义

这个因子试图捕捉一种 **“抗跌性”或“下跌中的异质性”** 。

- **它的核心可能是一个“偏度”或“非对称波动”因子** 。它惩罚那些在下跌日跌幅“异常大”（相对于自身历史波动）的股票，而奖励那些在下跌日中跌幅相对温和，或者在上涨日（无论涨多少）的股票。
- **另一种解读** ：因子在上涨日给出一个恒定的正信号（波动率本身），在下跌日则给出一个与下跌程度成比例的负信号。因此， **因子的最终值可以看作是一个经过波动率调整的“惩罚-奖励”得分** 。
  - 得分高的股票：要么经常上涨，要么在下跌时非常抗跌。
  - 得分低的股票：经常下跌，且在下跌时跌幅显著。
- **潜在的交易逻辑** ：做多因子值高的股票，做空因子值低的股票。这意味着买入那些“涨得普通但跌得少”或“涨跌都跟随大盘但下跌温和”的股票，避开那些“涨得普通但跌得狠”的股票。

**与传统指标的联系** ：

- 与 **夏普比率** 有相似思路（收益/波动），但它是日度的、非对称的。
- 与 **下行风险** 或 **索提诺比率** 的概念更接近，它特别关注下跌日的表现。
