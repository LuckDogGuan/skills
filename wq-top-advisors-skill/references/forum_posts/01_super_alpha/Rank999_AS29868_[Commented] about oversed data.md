  - Instead of repeatedly using the same few fundamental fields (like P/E ratio or ROE), spread usage across different metrics (e.g., debt ratios, cash flow, asset turnover).
  - This reduces the load on any specific field, avoiding the “overused” flag.

  - Mix in sentiment, technical indicators, or market microstructure data (like order book imbalance, tweet volume, etc.).
  - Even with >30% usage of fundamental data, the system treats diversified usage more leniently.
- **Rotate Data Field Combinations:**
  - Avoid using the same set of fields across multiple alphas. Create field combinations that are unique to each alpha batch.
- **Data Usage Monitoring:**
  - Maintain a simple log of which data fields are heavily used. If you spot patterns of overuse, adjust future alphas accordingly.
