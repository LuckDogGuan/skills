## `brain-data-scope`  技能分析

### 是什么

一个 **本地 SQLite 数据库查询工具** ，为 WorldQuant BRAIN 平台提供离线数据分析。它预打包了一个 257MB 的 SQLite 数据库（含 31,474 条 field report 和 160 个 dataset 组合），无需调用 BRAIN API 即可查询：

操作
用途

 `available_star` 
检查某个 dataset+region+universe+delay 组合是否存在

 `osis_badge` 
OS/IS Sharpe 比率、颜色徽章（绿/黄/红）、对比均值

 `neutralization_popup` 
中性化桶分布：每种中性化方式的 count/sharpe/percentage

 `field_report` 
单个 field 的覆盖率、pos/neg 比、偏度、峰度、年度趋势

### 测试结果（四个操作全部通过）

**1. Available Star**  —  `analyst11 / EUR / TOP1200 / Delay1`  返回  `★★★` （精确命中存在）

**2. OS/IS Badge**  —  `analyst11 / EUR / Delay1`

- OS/IS Sharpe:  **0.286** （1098个field）
- 均值: 0.478，低于均值 → 黄色 ( `#f9a825` )
- 统计截至: 2026-02-19

**3. Neutralization Popup**  —  `abnormal_return_earnings_release / USA / Delay1`

- 最佳中性化方式:  **STATISTICAL** （28.57%，sharpe 0.74）
- 其次: SUBINDUSTRY（21.43%）、INDUSTRY + SLOW（各 14.29%）
- 最高单桶 sharpe: SLOW_AND_FAST（2.34，但仅 7.14%）

**4. Field Report**  —  `analyst11 / EUR / anl11_2_1e`

- 覆盖率:  **81.74%** （1087.9/1200）
- pos_ratio: 100%（全正），neg_ratio: 0%
- 无偏度/峰度数据

### 实际用途

在 BRAIN 日常工作流中，这个技能可以帮你：

- **选数据集前** ：快速对比不同数据集的 OS/IS Sharpe 和中性化表现
- **选 field 前** ：看 field 覆盖率、pos/neg 比是否健康
- **中性化决策** ：看历史数据中哪种中性化方式对该数据集最有效
- **不消耗 API quota** ：全部离线查询

### 注意事项

数据库截止日期是  **2026-02-17/19** ，不是实时数据，但作为离线参考已经足够。
