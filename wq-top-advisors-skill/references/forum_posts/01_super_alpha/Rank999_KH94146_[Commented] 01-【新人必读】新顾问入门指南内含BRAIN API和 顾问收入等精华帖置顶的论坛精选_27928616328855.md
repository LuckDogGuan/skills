### 评论 #1 (作者: WL13229, 时间: 10个月前)

# 简读版本

本手册旨在为新顾问提供在 WorldQuant BRAIN 平台上入门所需的基本信息和资源。

## 1. 顾问之旅

本节概述了成为一名成功的 BRAIN 顾问的路径。

### 从用户到顾问

成为顾问的第一步是通过在挑战中累积 10,000 点积分来达到黄金级别。一旦你达到这个里程碑，你将收到申请顾问计划的邀请。

### 申请流程

申请流程包括填写表格、签订合同和接受背景调查。请务必仔细阅读所有说明，以免延误。

### 最初的 90 天

成为顾问的头三个月是“试用期”。在此期间，您的每日付款会较低，但这是学习窍门并建立您的 Alphas 产品组合的绝佳机会。

### 升级

试用期过后，您的“价值因子”将根据您的表现进行更新。更高的价值因子将带来显着更高的每日付款。

## 2. BRAIN API

本节介绍使用 BRAIN API 的要点，从初始设置到高级技术。

### 入门

- **Python 安装与学习：**  扎实的 Python 基础至关重要。确保您安装了最新版本的 Python 并熟悉基础知识。
- **API 文档：**  官方 BRAIN API 文档是您最好的朋友。它提供了有关所有可用功能以及如何使用它们的详细信息。您可以从 BRAIN 平台的“学习”部分下载它。 *注意：您必须是正式顾问才能访问 API 文档。*

### API 工作流程

使用 BRAIN API 的典型工作流程如下：

1. **认证：**  第一步是验证您的会话。这通过向  `/authentication`  端点发送带有您的用户名和密码的 POST 请求来完成。成功的身份验证将返回一个 JSON Web 令牌 (JWT)，您将用它来进行后续请求。
2. **数据探索：**  在构建 Alpha 之前，您需要了解哪些数据可用。
   - **获取数据集：**  使用  `/data-sets`  端点获取给定地区、总体和工具类型可用的数据集列表。
   - **获取数据字段：**  一旦有了数据集，您就可以使用  `/data-fields`  端点获取该数据集中数据字段的列表。
3. **回测：**  一旦有了 Alpha 表达式，就可以使用  `/simulations`  端点运行回测。
   - **运行回测：**  向  `/simulations`  端点发送带有您的 Alpha 设置和表达式的 POST 请求。成功的请求将返回一个 URL，您可以使用该 URL 检查回测的进度。
   - **检查回测状态：**  使用上一步中的 URL 检查回测的状态。如果回测仍在运行，响应将包含一个  `Retry-After`  标头，告诉您在再次检查之前要等待多长时间。
   - **获取回测结果：**  回测完成后，您可以使用相同的 URL 获取结果，包括 Alpha ID。
4. **Alpha 分析：**
   - **获取 Alpha 详细信息：**  使用  `/alphas/{alpha_id}`  端点获取有关您的 Alpha 的详细信息。
   - **获取 PnL：**  使用  `/alphas/{alpha_id}/recordsets/pnl`  端点获取您的 Alpha 的盈亏 (PnL) 数据。
   - **检查相关性：**  使用  `/alphas/{alpha_id}/correlations/self`  和  `/alphas/{alpha_id}/correlations/prod`  端点检查您的 Alpha 与您的其他 Alpha 以及与生产中的 Alpha 的相关性。

### 关键 API 功能

- **自动化 Alpha 创建：**  BRAIN API 允许您自动化创建和测试 Alphas 的过程。这是一个强大的功能，可以显着加快您的研究速度。
- **错误处理：**  熟悉使用 API 时可能发生的常见错误。论坛中有大量关于此主题的信息。
- **高级技术：**  浏览论坛，获取经验丰富的顾问关于如何充分利用 API 的提示和技巧。这包括从优化代码到使用您可能不知道的高级功能的所有内容。

### 资源

- **云计算：**  学习如何设置和使用 AWS、Google Cloud 等云计算服务来运行您的代码和回测。
- **AI 驱动的助手：**  了解如何使用大型语言模型 (LLM) 和其他 AI 工具来协助您的研究和编码。

## 3. Alpha 开发

本节提供有关如何开发高质量 Alphas 的指南。

### 怎样才算一个好的 Alpha？

- **不仅仅是夏普比率：**  虽然高夏普比率是可取的，但它不是唯一重要的指标。一个好的 Alpha 还应该具有低换手率、高利润率和稳定的 PnL。
- **数量与质量：**  一开始，更重要的是专注于提交足够数量的 Alphas 来建立您的投资组合。随着您获得更多经验，您可以开始更多地关注提交的质量。一个好的经验法则是每月至少提交 40 个 Alphas。
- **低换手率是关键：**  平台会惩罚换手率高的 Alphas，因为它们的交易成本更高。目标是换手率低于 30%，理想情况下低于 15%。
- **子 universe 表现很重要：**  您的 Alpha 不仅应在主 universe 中表现良好，还应在较小的子 universe 中表现良好。这表明您的 Alpha 是稳健的，而不仅仅是侥幸。
- **经济意义的重要性：**  一个好的 Alpha 应该具有明确的经济学原理。在开始编码之前，花点时间思考一下为什么您的 Alpha 应该起作用。为您的 Alpha 编写详细的描述是充实您的想法的好方法。

### 提交什么样的 Alphas？

- **平衡数量与质量：**  一开始，专注于提交足够数量的 Alphas 来建立您的投资组合。随着您获得更多经验，您可以开始更多地关注提交的质量。一个好的经验法则是每月至少提交 40 个 Alphas。
- **注意您的换手率：**  保持您的换手率低于 70%，理想情况下对于有经验的顾问来说低于 30% 甚至 15%。
- **在子 universe 中进行测试：**  您的 Alphas 不仅应在主 universe 中表现良好，还应在较小的子 universe 中表现良好。这表明您的 Alpha 是稳健的，而不仅仅是侥幸。
- **寻找稳定的 PnL：**  您的 Alpha 的 PnL 应该是一条稳定的、向上倾斜的线。这表明您的 Alpha 具有持续的盈利能力。
- **使您的投资组合多样化：**  多样化的 Alphas 投资组合更稳健，更不容易受到市场低迷的影响。
- **具有经济意义：**  您的 Alphas 应该具有明确的经济学原理。在开始编码之前，花点时间思考一下为什么您的 Alpha 应该起作用。

### 常见陷阱及如何避免

- **时区混淆：**  BRAIN 平台使用多个时区。请务必熟悉它们，以避免在提交 Alphas 和检查结果时出现混淆。
- **数据集选择不佳：**  并非所有数据集都是平等的。寻找在质量和数量上都有良好平衡的数据集。
- **忽略金字塔：**  金字塔是 BRAIN 生态系统的重要组成部分。不要只关注美国地区；探索其他地区以寻找新的机会。
- **误解天才计划：**  天才计划是赚取额外收入的好方法，但了解排名标准很重要。不要只专注于提交大量的 Alphas；专注于提交高质量、独特的 Alphas。
- **提交低质量的 Alphas：**  提交一个高质量的 Alpha 比提交四个低质量的 Alphas 要好。专注于创建具有高夏普比率和低换手率的 Alphas。
- **忽略您的基本工资：**  您的基本工资由您的价值因子决定。要增加您的基本工资，请专注于通过提交高质量、不相关的 Alphas 来提高您的价值因子。
- **检查速度慢：**  检查过程可能很慢，但有办法加快速度。例如，您可以使用本地缓存来存储数据，避免每次运行回测时都重新下载。
- **不当使用负号：**  负号是一个强大的工具，但正确使用它很重要。请务必阅读文档以了解如何在表达式中使用负号。

## 4. 顾问收入

了解您如何获得报酬至关重要。本节将分解顾问薪酬的不同组成部分。

### 了解您的收入

- **Alpha 类型：**  了解四种主要 Alpha 类型（RA、ATOM、PPAC、SA）之间的区别，以及它们如何为您的收入做出贡献。
- **天才计划：**  天才计划为表现出色的顾问提供了赚取可观季度付款的机会。
- **每日报酬：**  了解您的每日付款是如何计算的，以及您可以做些什么来增加它们。
- **付款结构：**  清楚地了解整体付款结构，包括基本工资、季度奖金、推荐奖金和比赛奖金。
- **价值因子：**  价值因子是影响您收入的关键指标。了解它是什么以及如何改进它。

### 增加您的每日报酬

- **专注于一个领域：**  不要试图成为万事通，而应专注于特定地区或数据集。该帖子建议从 ASI（亚洲）地区开始，因为它有大量的 Alphas 可供发现，并且有更高的机会让您的 Alphas 加权。
- **解锁超级 Alphas：**  您的首要目标应该是提交 100 个常规 Alphas 以解锁超级 Alpha 功能。超级 Alphas 是增加您收入的好方法，因为它们允许您将现有的 Alphas 组合成更强大的策略。
- **提高您的价值因子：**  价值因子是决定您每日报酬的关键指标。它每月更新一次，并基于您过去三个月的 Alphas 表现。要提高您的价值因子，请专注于提交与您现有 Alphas 或生产 Alphas 不相关的高质量 Alphas。
- **利用主题：**  主题是提供质量得分乘数的特殊活动。只要有可用的主题，请务必参加，因为它们可以显着增加您的收入。

- **经济意义的重要性：**  一个好的 Alpha 应该具有明确的经济学原理。在开始编码之前，花点时间思考一下为什么您的 Alpha 应该起作用。为您的 Alpha 编写详细的描述是充实您的想法的好方法。

总结几点：

1. 不仅要鲁棒，还要具备经济性，有投资价值；月球固然好，当前没有投资价值，因此没人去；

2. 子universe：重点在流动性，如果只是流动性差的票占高比例，有何意义呢？

3. 经济学意义：无根之水不能长久，没有任何经济学意义的组合，大概率是过拟合的。
