如何将全部alpha下载到本地，并进行数据分析，这是通往高手的必经之路。这里分享下技术实现细节。

任何需求，其实都是对wq api的理解，本次的api主要是获取alpha api：

[https://api.worldquantbrain.com/users/self/alphas](https://api.worldquantbrain.com/users/self/alphas)

题外话：之前培训老师给的api的时区是 -04:00, 经查阅资料，美国东部时区为 -05:00,这次代码给修正了。目前过滤条件的 "><="符号不用写 %3E, %3C了， 可以直接写 >=, <=。

之前有小伙伴在微信群里问到，为什么下载的最大值是10000，这里猜测下，根据多年的后端经验，这里的10000行数据是给网页展示的，没必要给出太多。 经过对此api的分析发现 `dateCreated` 参数是日期类型，在网页版里，前端代码已经卡死了，最细粒度就是 天/day， 所以1天内超过1w个数据，就不会被显示。

所以这里只需要更改freq即可， freq在金融领域用的比较多，或者说在pandas里用的比较多，即数据的采样频率，pandas的freq可选项很多，比如D（天）, M（月）,S（秒）,30S（30秒），T(分钟)等等等等，有很多，感兴趣的同学可以查阅官方文档。我们可以将freq改为更细粒度的，比如小时，也就是说在一个freq周期内扫描一次，粗略估计了下，一天4万条，每次上线是1万的话，大概6小时一个freq比较合适，为了保险点，也可以设置为更短些，比如4个小时：

period = pd.DateOffset(hours=4)

down_alpha.py

```
import requestsimport pandas as pd​# 你的alpha回测开始时间， 尽量设置顾问alpha开始提交的时间，学徒的没啥不要下载了start_date = '2024-12-07'​​def login():​    username = "xxxx"    password = "xxxx"​    # Create a session to persistently store the headers    s = requests.Session()​    # Save credentials into session    s.auth = (username, password)​    s.post('https://api.worldquantbrain.com/authentication')    return s​​def get_alpha_unsubmitted(date):​    # N小时一个作为一个取数区间    period = pd.DateOffset(hours=1)    tomorrow = date + pd.DateOffset(days=1)    tomorrow = tomorrow.isoformat()​    start_date = date.isoformat()    end_date = (date + period).isoformat()​    all_data = []    while start_date != tomorrow:        # 步长最大是100， 即便设置成500，返回的 next 也会强制改为100，此举为服务器行为        step = 100        offset = 0        s = login()        # 按小时计算​        while True:            url = f"https://api.worldquantbrain.com/users/self/alphas?limit={step}&offset={offset}" \                "&status=UNSUBMITTED%1FIS_FAIL" \                f"&dateCreated>={start_date}-05:00&dateCreated<{end_date}-05:00" \                "&settings.region=USA"            print(url)            response = s.get(url)            try:                data = response.json()            except BaseException:                # 这里可能需要查看下错误                # 比如用ipdb                # import ipdb; ipdb.set_trace()                s = login()            count = data['count']            if count == 0:                break            print(f'{start_date} -> {end_date}, count:', count)            results = data['results']            df = pd.DataFrame(results)            all_data.append(df)​            next_url = data['next']            # 本轮结束，进行下一轮            if next_url is None:                break            # 更新 offset            offset += step​        start_date = end_date        end_date = pd.to_datetime(end_date) + period        end_date = end_date.isoformat()        print(f'update start_date: {start_date}, end_date: {end_date}===============')    df = pd.concat(all_data)​​def main():​    now = pd.Timestamp.today().strftime('%F')​    days = pd.date_range(start_date, now)    for date in days:        get_alpha_unsubmitted(date)​​if __name__ == "__main__":    ''' python down_alpha.py '''    main()
```
