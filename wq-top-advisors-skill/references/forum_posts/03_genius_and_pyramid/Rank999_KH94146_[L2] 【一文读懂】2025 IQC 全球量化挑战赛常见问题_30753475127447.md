## **一、规则与参赛资格**

### **1.1 关键时间节点？**

- 2025年3月18日：报名和团队组建开始。
- 2025年3月18日：第一阶段资格赛提交开始。
- 2025年5月14日：报名和团队组建结束。
- 2025年5月19日：第一阶段资格赛提交结束。
- 最晚2025年5月26日：晋级第二阶段IQC的团队将通过电子邮件收到通知。
- 2025年5月27日：第二阶段Alpha提交开始。
- 所有截止时间均为东部时间23:59。

### **1.2 参赛资格**

- 要参加比赛，你和你的队友必须首先注册成为 WorldQuant BRAIN 用户，并获得登录凭证。截至国际量化竞赛（IQC）开始时（2025年3月18日东部时间00:01）或在竞赛期间的任何时候，以下人员不符合参赛资格：
- - 截至2025年3月17日，作为 WorldQuant 的 BRAIN 研究顾问（包括已获得有条件批准的人员）（由 WorldQuant 根据其唯一判断确定）；
- - 曾经是 VRC 或 WebSim 的顾问；
- - WorldQuant 大学的学生或校友；
- - 曾获得2022年全球阿尔法竞赛和/或由 WorldQuant 组织的任何先前国际量化竞赛的奖项（包括与之相关的任何个人）（由 WorldQuant 根据其唯一判断确定）；
- - WorldQuant Predictive Technologies LLC（或其关联公司）的现任员工或顾问/承包商；
- - WorldQuant, LLC（或其任何关联公司）的现任或前任员工或顾问/承包商（除非已在本条款中另行说明）。
- 请参阅 IQC 指南以了解更多信息和限制。

### **1.3 我是否必须是在校生？**

根据 IQC 指南的第 7 条，大学的教职员工或校友可以参加：

团队可以由在校学生、教职员工或校友组成，然而，每个团队成员必须与同一所大学有关联。

如果你尚未注册入学某所大学，那么在没有大学名称的情况下，你将没有资格参加 IQC。不过，你可以参加 WorldQuant 挑战赛，而不是参加 IQC。

## **二、资格赛第一阶段**

### **2.1 第一阶段分数如何计算？**

**IQC 第一阶段分数由以下两个部分组成：**

> **1. 个人资格赛分数 Individual Qualifier Score: ：**

- 你的分数基于你在平台上提交的阿Alpha法的数量和质量（在 5 年样本期内的表现）。
   - 你的分数还取决于当天其他用户提交Alpha的数量和质量。
   - 分数是按天（东部标准时间）计算的，而不是按每个阿尔法计算。排行榜每天东部标准时间上午 3 点刷新。
   - 你每天能获得的最高分数是 2000 分。通常，这涉及到每天提交 1 到 3 个阿尔法。
   - 没有负分。你的分数不会减少。
   - 这是一个个人分数，而不是团队分数。
   - 每个Alpha的分数取决于多种因素，包括以下几点：
     - Alpha的类型（较小的Universe可能获得更高的分数）。
     - Alpha的质量（例如，更高的Fitness，更低的自相关性 Self Correlation）。
   - 每个合格的Alpha是根据其与其他参与者当天提交的Alpha的表现进行评分的。
   - 当至少有一个团队成员达到 10000 个资格赛分数时，团队在 IQC 排行榜上解锁Merge Performance：

> **2. Merge Performance Score：**

- 当第一个团队成员达到 10000 Individual Qualifier Score时，这个分数才会出现在排行榜上（作为“IS 分数”列及其子分数“D0 分数”和 D1 分数）。

- 在 IQC 第一阶段，团队将根据相应的Merge Performance而进行的计算结果进行排名。团队的“Merged Performance Series”是通过将团队的alpha按其P&L以等权重(weight)方式合并计算得出的，即每个alpha投资的金额相等。

- 每天没有限制可以获得的分数。
   - 所有成员的Alpha都被考虑用来计算合并表现分数，包括那些提交用于Individual Qualified Score的Alpha。
   - 如果新的Alpha没有为你的现有Alpha Pool增加价值，新的Alpha提交可能会降低你的Merge Performance Score。

> **第一阶段最终分数是如何计算的？**
> **合并表现 = IS 分数的 25% + OS 分数的 75%**

IS 分数 = D1 分数 + (D0 分数)/3

- 25% 基于你的Alpha在 5 年样本期内(IS)的表现。这个分数在第一阶段开始时在排行榜上可见。
- 75% 基于你的Alpha在 1 年样本外(OS)期间的表现。这个分数将在第一阶段结束时在排行榜上可见。

由于 OS 分数只有在第一阶段结束时才可见，请记住，为了提高样本外表现，要关注想法和实施质量，而不仅仅是表现数字和指标：清晰的想法和稳健的实施通常会在 OS 期间保持强劲，而表现数字可能会发生变化。我们将继续在直播网络研讨会和社区中分享更多技巧。

> **影响第一阶段分数的指标有：**

- Merge Performance Series的夏普比率：夏普比率越大，分数就越高，其他参数保持不变。
- Merge Performance Series的回报与回撤比率：这个比率越大，分数就越高，其他参数保持不变。
- Turnover。这个参数越大，分数就越差，其他参数保持不变。在其他参数相同的情况下，将周转率降低到 12.5% 以下，可能并不一定能提高你的分数。

**我可以在哪里看到Merge Performance指标？**

如果你达到了 10000 分 Individual Qualified Score，那么在Simulation屏幕右侧的Result部分，向下滚动到Performance Comparison部分并点击下拉箭头。

### 2.3 排行榜的更多说明

| 排名 | 基于 IS 分数的值 |
| 资格赛分数 | 第一阶段资格赛分数。最高个人分数 = 10000 分，这将解锁团队的合并表现分数的组成部分：IS 分数、D0 分数和 D1 分数值 |
| IS 分数 | 基于 5 年样本期内（2018 - 2023）阿尔法的表现的样本内表现分数。D1 分数 + (D0 分数)/3。占合并表现分数的 25% 权重 |
| D0 分数 | 团队合并 D0 阿尔法池的样本内表现分数。是合并表现的 IS 分数的组成部分 |
| D1 分数 | 团队合并 D1 阿尔法池的样本内表现分数。是合并表现的 IS 分数的组成部分 |
| OS 分数 | 样本内表现分数。占合并表现分数的 75% 权重。将在 IQC 的每个阶段结束揭晓

### 2.4 第一阶段的alpha和积分是否带入第二或第三阶段？

是的

### 2.5 第二阶段的晋级比例？

所有在 IQC 排行榜上解锁了Merge Performance Score访问权限的团队，将有资格参加第二阶段。为此，团队中至少需要有一名成员获得 10000 分 Individual Qualify Score。

在往年比赛中，大约全球排名前 20% 的团队（根据合并表现的 IS 和 OS 分数），从第一阶段晋级到第二阶段。我们预计在这一版本中会有类似的比率。

10000 分Individual Qualifed Score对应的Merge Performance Score，足以使任何团队在世界排名中稳居前 20%，即使包括在第一阶段结束时揭晓的样本外分数也是如此。因此，你的第一个目标是在第一阶段获得 10000 个资格赛分数。在达到这个目标之后，将注意力转移到Merge Performance Score上，因为这是决定后续阶段成功的关键指标。

### **2.6 Merge Performance Score 与提交Alpha的关系？**

合并表现的 Sharpe 比率。在其他参数相同的情况下，Sharpe 比率越大，分数就越高。
合并表现的回报与回撤比率：在其他参数相同的情况下，该比率越大，分数就越高。
合并表现的 Turnover。在其他参数相同的情况下，该参数越大，分数就越低。在其他参数相同的情况下，将 Turnover 降低到 12.5% 以下，可能并不一定能提高你的分数。

作为一个经验法则，表现更好的阿尔法会带来更好的合并表现。我们所说的“表现更好”，指的是高 Sharpe 比率、高回报、高 Fitness、低 Turnover 和低相关性。

阿尔法的数量：具有低相关性的阿尔法数量越多，可以增加合并表现。大量高相关性的阿尔法可能会降低你的合并表现。

### 排行榜的更新频率？

IQC第一阶段资格分数每天刷新一次，大约在东部时间凌晨3点。如果你已经达到了可能的最大资格分数10,000分，你还可以访问合并表现指标，该指标每2小时刷新一次。

### 每日提交和得分上限？

在IQC第一阶段，直到你达到最大个人分数10,000资格积分之前：

- 每个人每天能够达到的最高分数是2,000分。通常，这涉及每个人每天提交1到2个阿尔法。
- 分数是按天计算的（东部时区），而不是按阿尔法计算的。
- 没有负分。你的分数不会减少。

在你达到10,000资格积分之后，你将专注于合并表现分数：

- 每天可以赚取的合并表现分数没有限制。
- 新的阿尔法提交可能会降低你的合并表现分数。

## **三、IQC顾问收入**

**3.1 参加IQC是否自动获得顾问？**

不是，需要完成顾问申请流程，包括初审、复审和合同签订。当前排队人数较多，更高的提交将有助于更快地获批

**3.2 什么时候开始获得收入？**

在正确提交顾问合同后，您的有条件顾问通常在1-2天内可能获批。获批后即可积累每日的顾问收入。在此之前是没有办法获得收入的。

**3.3 IQC提交的Alpha是否计入初审、复审的优先排队序列？**

会的，无需单独申请，后台会每周计算高优序列同步至审核团队

**3.4 在哪里查看我的每日收入？**

平台右上角我的--> Performance

**3.5. IQC比赛结束后是否会自动获得正常顾问的资格？**

是的，预计在7月中您会转为正常顾问，获得全部顾问权限和数据、操作符、genius 等

**3.6 IQC顾问的收入何时发放？**

与正常顾问的周期一致，Base Payment每两月发放一次，Quarterly Payment每季度发放一次。值得注意的是，您需要成为Full Consultant，即完成Background Check后收到邮件要求你正确填写银行卡信息后，方可开始收款，在此之前您的待付款金额会持续累计。最低付款金额为10USD起。更多内容请阅读顾问中文论坛内相关帖子或FAQ搜索Payment Date等内容
