## 帖子正文

大家好，我是去年10月10日注册worldquant平台，10月19日参加的线下培训，但我直到今年2月初才拿到顾问权限，在拿到顾问权限后，自己一开始是按照给的模板跑的USA数据集，但连续很多天没有任何产出，这时候自己才意识到按照上课给的模板，一字不改是很难跑出来任何结果的，因为大部分容易跑的模板别人都已经跑过了，这之后最好的方法就是老老实实看论坛，看其他人的方法，对照着论文，适当修改模板。

一开始自己是单进程跑的，后来意识到课程给的代码里的multi_simulate方法并不是真正的并发，然后自己做了简单的修改，以最小的变动提供了并发，变动如下

from multiprocessing import Process, Pool

# Load alphas to task pools
fo_pools = load_task_pool(fo_alpha_list, 10, 9)
print(fo_pools[0])

pool = Pool(10)
for i, i_pool in enumerate(fo_pools):
    print(f"now is continue, now index is {i}")
    pool.apply_async(multi_simulate, ([i_pool], "SUBINDUSTRY", "ASI", 'MINVOL1M', 0))
pool.close()
pool.join()

通过这种变动可以用最小的变动实现多进程，适合懒人操作。

目前这种方法每天可以支持大概2W左右回测，实际上前期基本上每天回测数都在3W以上，但最近不知道什么原因，回测降到了2W左右。
