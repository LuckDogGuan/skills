## 帖子正文

## 1、DCC 是什么？

DCC 是一项仅限个人参加的竞赛，参赛者使用 Python 和 BRAIN API 直接生成基于新闻的数据字段，并在此基础上构建 Alpha 策略。

## 2、参赛资格与规定

- 名额限制：前 3000 名报名的顾问和有条件顾问有资格参加
- 参赛形式：仅限个人，不允许组队

## 3、如何参赛

- 标记方式：竞赛 Alpha 无需提交，只需标记为 "DCC2026" 即可
- 时间窗口：仅在官方竞赛期间（2 月 23 日 – 4 月 5 日）模拟的 Alpha 才会被纳入评分
- 数据要求：Alpha 必须使用自建的数据字段和支持字段

## 4、评分机制

### 4.1 综合表现得分（Merged Performance Score）

基于所有标记模拟的合并表现，采用 IS 与 OS 结合评估：

```
指标            权重   说明 样本内（IS）表现    25%    每6小时更新样本外（OS）表现    75%    竞赛结束时更新
```

**评估指标：**

- 夏普比率（越高越好）
- 收益回撤比（越高越好）
- 换手率（越高越受惩罚）

**公式：总得分 = 0.25 × IS 得分 + 0.75 × OS 得分**

### 4.2 评估得分（Assessment Score）

- 适用对象：总得分排名前 50 且符合资格的参赛者
- 评估内容：笔记本、演示文稿、研究方法（清晰度、结构、原创性、契合度）
- 公式：最终得分 = 0.5 × 综合表现排名 + 0.5 × 评估排名

## 5、排名资格要求

要获得排名资格，必须满足：

- 标记至少  **10 个模拟**
- 创建至少 **1 个数据字段**

## 6、取消资格情形

若笔记本或演示文稿出现以下情况，可能被判定不合格或取消资格：

- 无效或未能反映实际研究过程
- 不完整或质量不足
- 不符合输入输出格式规范：
  - 输入：startdate、enddate
  - 输出：created datafield、parquet 文件

## 7、奖金设置

```
名次          奖金 第 1 名       $3,000 第 2 名       $2,500 第 3 名       $2,000 第 4–10 名   $1,000 第 11–25 名  $350 第 26–50 名  $200 
```

**特别奖励** ：除总排名前 50 外，每个国家前 3 名可额外获得 $100

**总奖金池** ：$30,000
