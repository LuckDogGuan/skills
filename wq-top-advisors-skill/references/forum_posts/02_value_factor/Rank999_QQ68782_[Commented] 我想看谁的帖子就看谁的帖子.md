## 帖子正文

怎么看某个大佬的所有的帖子?

在论坛可以点击ta的头像, 进入ta的论坛个人主页, 在个人主页里可以看到所有发的帖子;

比如游戏王的主页是: 
 [https://support.worldquantbrain.com/hc/en-us/profiles/26858512793111-worldquant brain赛博游戏王](https://support.worldquantbrain.com/hc/en-us/profiles/26858512793111-worldquant brain赛博游戏王)

如何找到任意大佬的主页地址呢? 能通过brain userId(比如worldquant brain赛博游戏王)找到论坛主页吗?

其实是可以的, 这个接口可以实现;

登录brain后, 在浏览器打开: 
```
 [https://support.worldquantbrain.com/hc/api/internal/communities/mentions.json?query=worldquant brain赛博游戏王](https://support.worldquantbrain.com/hc/api/internal/communities/mentions.json?query=worldquant brain赛博游戏王) 
```
query后面的参数是userid

返回结果:
```
[
{"id":26858512793111,"name":"worldquant brain赛博游戏王",
"avatar_url":" [https://secure.gravatar.com/avatar/c8fa0a8e0a7a6636a99080189831b826?default=https%3A%2F%2Fassets.zendesk.com%2Fhc%2Fassets%2Fdefault_avatar.png\u0026r=g](https://secure.gravatar.com/avatar/c8fa0a8e0a7a6636a99080189831b826?default=https%3A%2F%2Fassets.zendesk.com%2Fhc%2Fassets%2Fdefault_avatar.png\u0026r=g) ",
"user_profile_path":"/hc/profiles/26858512793111-worldquant brain赛博游戏王"
}]
```
其中的 user_profile_path 就是个人主页地址, 拼接到  [https://support.worldquantbrain.com](https://support.worldquantbrain.com)  后面就是完整的地址
