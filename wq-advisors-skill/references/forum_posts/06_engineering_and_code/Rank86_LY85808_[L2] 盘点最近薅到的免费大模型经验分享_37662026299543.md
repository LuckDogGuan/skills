**Opencode**

需要安装oh-my-opencode插件，这个是神器！有它和没它，opencode是2种产品。安装这个活儿，可以交给opencode。opencode安装后，有那几个免费模型，可以直接帮你做这些脏活儿累活。

如果本地已有claude code，opencode会自动加载claude code的plughin和skills。但是，大模型还是需要自己添加的。

目前，GLM4.7-free，用的人太多，会不断有报错提示调用过多。minmax用的人少，可以随便跑。

opencode最近才开始火，更新非常频繁，也有一些bug。网上有人说，非程序员慎用。它的确会 偶有莫名其妙的错误。

### 评论 #2 (作者: ZZ44620, 时间: 5 months ago)

补充一个，
nvidia也有送的模型，是deepseek GLM kimi这些，然后免费使用api，可以去 [https://build.nvidia.com/models](https://build.nvidia.com/models)  ，注册，如果qq邮箱不支持可以换谷歌邮箱，后面绑定手机是支持+86的，另外限制是 Up to 40 rpm，也就是40分钟最多请求40次。
我们注册完绑定手机之后就可以申请api，然后使用方法也是openai兼容，在文档中
以deepseek为例，在文档 [https://docs.api.nvidia.com/nim/reference/deepseek-ai-deepseek-r1-infer](https://docs.api.nvidia.com/nim/reference/deepseek-ai-deepseek-r1-infer)

右边侧栏有example能看到model和base url ( [https://integrate.api.nvidia.com/v1](https://integrate.api.nvidia.com/v1) )

> import requests
> url = " [https://integrate.api.nvidia.com/v1/chat/completions](https://integrate.api.nvidia.com/v1/chat/completions) "
> payload = {
> "model": "deepseek-ai/deepseek-r1",
> "temperature": 0.6,
> "top_p": 0.7,
> "frequency_penalty": 0,
> "presence_penalty": 0,
> "max_tokens": 4096,
> "stream": False
> }
> headers = {
> "accept": "application/json",
> "content-type": "application/json"
> }
> response = requests.post(url, json=payload, headers=headers)
> print(response.text)
