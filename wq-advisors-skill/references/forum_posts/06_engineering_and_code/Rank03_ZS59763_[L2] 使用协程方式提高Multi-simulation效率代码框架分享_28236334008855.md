## 帖子正文

现有的模板 multi_simulate 函数是串行执行，

劣势是提交10组alpha，需要等10组alpha全部执行完成之后，才能进行下一组，若其中一组很快回测完成，要继续等剩下的9组alpha全部完成， 这样严重影响回测效率

有两种解决方式：

1.使用线程，使用线程池开10个线程，每个线程支持一组执行；

2.使用协程，开10个协程任务执行。

我是使用的第2种方式，也只做第2种方式的分享，因为个人感觉协程在IO密集型模型中是优于线程的，有使用第一种方式的同仁也欢迎分享；

multi simulate和check submission操作是互不干扰的，即运行10个回测的同时也可以同时进行3个 check submission，所以要尽可能的提高效率，要创建两种任务类型一种回测，另一种check

以下代码不能直接使用，目的在于分享这种方法

```
async def run(task: Task, alpha_list):    try:        # check 功能        check_worker_queue = Queue()        # check worker        check_workers = [            create_task(check_worker(check_worker_queue, _, task)) for _ in range(3)        ]        multi_simulate_worker_queue = Queue()        task_pool = load_task_pool(alpha_list, 10, 10)        # 加入任务队列，worker执行        for index, item in enumerate(task_pool):            # 辅助跳过条数设置            if task.skip_count > index:                continue            # 创建一组10个alpha            data = multi_simulate_data(task, item)            # 加入回测队列            multi_simulate_worker_queue.put_nowait((index, data))        # 回测worker        workers = [            create_task(                worker(multi_simulate_worker_queue, _, check_worker_queue, task)            )            for _ in range(10)        ]        # 收尾 multi_simulate worker        # 等待直到队列处理完毕。        await multi_simulate_worker_queue.join()        logger.info("multi_simulate_worker_queue 队列处理完毕")        # 取消我们的工作任务。        for item in workers:            item.cancel()        # 等待直到所有工作任务都被取消。        await asyncio.gather(*workers, return_exceptions=True)        logger.info("multi_simulate_workers 已关闭")        # 收尾 check_worker        # 等待直到队列处理完毕。        await check_worker_queue.join()        logger.info("check_worker_queue 队列处理完毕")        # 取消我们的工作任务。        for item in check_workers:            item.cancel()        # 等待直到所有工作任务都被取消。        await asyncio.gather(*check_workers, return_exceptions=True)        logger.info("check_workers 已关闭")    except Exception as e:        logger.exception(e)# 回测worker实现，check worker 也是相同的实现async def worker(task_queue: Queue, num: int, check_worker_queue: Queue, task: Task):    task.logger.info(f"multi simulate worker {num} ready")    while True:        index, data = await task_queue.get()        try:            simulate_list = await multi_simulate(task, index, data)            if simulate_list:                for process_id in simulate_list:                     # 回测worker完成之后，加入check worker 队列                    check_worker_queue.put_nowait(process_id)        except Exception as e:            task.logger.exception(e)        finally:            task_queue.task_done()
```
