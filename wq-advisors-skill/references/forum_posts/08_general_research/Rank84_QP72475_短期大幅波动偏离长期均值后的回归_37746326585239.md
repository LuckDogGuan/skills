## 帖子正文

```
Alpha027 = 
rank((sum(100 * (close - delay(close, 1)) / close, 20) / 20) 
- ((100 * (close - delay(close, 1)) / close) * 
   vwap / (sum(vwap, 20) / 20)))^2
```

### 核心分解：

1. **第一部分：长期平均收益率**
   ```
   avg_return_20 = mean(100 * (close_t - close_{t-1}) / close_t, 20)
   ```
   计算过去20日平均百分比收益率（以当日收盘价为分母）
2. **第二部分：调整项**
   ```
   adjustment = (100 * ret_today) * (vwap_today / avg_vwap_20)
   ```
   其中：
   - `ret_today = (close_t - close_{t-1}) / close_t`
   - `avg_vwap_20 = mean(vwap, 20)`
3. **最终计算** ：
   ```
   factor_value = (avg_return_20 - adjustment)^2
   rank(factor_value)
   ```

## 二、经济含义与市场逻辑

### 核心逻辑：

1. **均值回归与动量结合** ：
   - 当短期价格变动（今日收益率）与20日均线方向一致，但幅度显著不同时产生信号
   - VWAP比率衡量的流动性异常起到调节作用
2. **流动性调节机制** ：
   - `vwap_today / avg_vwap_20 > 1` ：今日交易集中在较高价位，可能反映卖方压力
   - 该比率放大今日收益率的影响权重
3. **平方操作的目的** ：
   - 放大异常值的影响，使因子对极端情况更敏感
   - 确保输出始终为正，便于排序比较

### 市场现象捕捉：

- **过度反应后的反转** ：短期大幅波动偏离长期均值后的回归
- **流动性冲击** ：异常VWAP表明大单交易的影响
- **波动率聚集效应** ：平方项隐含着对波动率变化的响应
