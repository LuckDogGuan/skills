# 衡量了股票的长期相对强度

- **链接**: https://support.worldquantbrain.com衡量了股票的长期相对强度_40765139378327.md
- **作者**: 顾问 QP72475 (Rank 84)
- **发布时间/热度**: 1个月前, 得票: 27

## 帖子正文

```
(-1 * sign((close - delay(close, 7)) + delta(close, 7))) * (1 + rank(1 + sum(returns, 250))))
```

这个公式看起来复杂，但我们可以将其分解为几个部分来理解。其中通常包含以下函数：

- `close` ：当日收盘价
- `delay(close, n)` ：n天前的收盘价
- `delta(close, n)` ：当前收盘价与n天前收盘价之差，即  `close - delay(close, n)`
- `sign(x)` ：符号函数，x>0返回1，x<0返回-1，x=0返回0
- `rank(x)` ：横截面上，对所有股票的x值进行排序，并转换为归一化的排名（通常在0到1之间）
- `sum(returns, n)` ：过去n天的收益率之和
- `returns` ：收益率，通常定义为  `(close / delay(close, 1)) - 1`

### 2. 因子逻辑解析

我们将公式拆解成两个核心部分来理解其经济含义。

#### 部分A：短期反转与动量确认

`(-1 * sign( (close - delay(close, 7)) + delta(close, 7) ))`

让我们仔细分析括号内的内容：

- `(close - delay(close, 7))` ：这就是过去7天的价格变化量。
- `delta(close, 7)` ： **同样也是**   `close - delay(close, 7)` 。

所以，这个表达式简化为：
 `sign( (7日价格变化) + (7日价格变化) )`  =  `sign( 2 * (7日价格变化) )`

由于 `2` 是正数， `sign` 函数的结果完全取决于 **过去7天的价格变化** 的方向：

- 如果过去7天股价上涨， `sign(...)`  = 1。
- 如果过去7天股价下跌， `sign(...)`  = -1。
- 如果持平，则为0。

然后，整个A部分乘以  `-1` ：

- **如果过去7天涨**  -> A =  `-1 * (1)`  =  `-1`
- **如果过去7天跌**  -> A =  `-1 * (-1)`  =  `1`

**部分A的逻辑解读：** 
这部分本质上是一个 **短期反转信号** 。

- 当股票在短期内（过去一周）上涨时，该部分产生一个负信号（-1），预示着未来可能下跌。
- 当股票在短期内下跌时，该部分产生一个正信号（1），预示着未来可能上涨。

#### 部分B：长期动量（或强度）修饰

`(1 + rank(1 + sum(returns, 250)))`

- `sum(returns, 250)` ：过去250天（约一年）的总收益率。这代表了股票的 **长期动量** 。
- `1 + sum(returns, 250)` ：加1是为了避免负值，保证后续计算有意义。
- `rank(1 + ...)` ：在全部股票池中，根据这个“1+年收益率”进行排名。排名越靠前（接近1），代表该股票过去一年的表现越好；排名靠后（接近0）则表现越差。
- `1 + rank(...)` ：再次加1，使得这个值域落在[1, 2]之间。

**部分B的逻辑解读：** 
这部分衡量了股票的 **长期相对强度** 。它是一个始终为正的修饰器，值越大，代表该股票在过去一年的表现越出色。

#### 整体逻辑：短期反转与长期动量的结合

整个因子的值是  **A * B** 。

- **对于过去一年表现很好的股票（B值大）：**
  - 如果它近期（7天）下跌（A=1），因子会给出一个 **强烈的正向信号** （大正数）。这意味着市场在抛售一支优质股，可能是错杀，是很好的买入机会。
  - 如果它近期上涨（A=-1），因子会给出一个 **强烈的负向信号** （大负数）。这可能意味着优质股的短期利好出尽或过热，提示卖出。
- **对于过去一年表现很差的股票（B值小）：**
  - 即使它近期下跌（A=1），因子的正向信号也较弱（小正数）。因为这支股票本身就很弱，近期的下跌可能是其衰弱的延续，反弹动力不足。
  - 如果它近期上涨（A=-1），因子的负向信号也较弱（小负数）。弱势股的反弹可能只是噪音，不值得大力做空。

**核心思想：** 
试图捕捉的是  **“强势股短期回调”的买入机会** 和  **“强势股短期冲高”的卖出机会** 。它认为，在长期强势的股票上，短期的价格反向波动提供了更好的交易时机。而对于长期弱势的股票，交易机会则不那么明确。
