# “尾盘弱势”或“价格压抑”的持续性效应

- **链接**: 尾盘弱势或价格压抑的持续性效应.md
- **作者**: 顾问 QP72475 (Rank 84)
- **发布时间/热度**: 6个月前, 得票: 30

## 帖子正文

```
rank(sma(vwap - close, 5) / stddev(close, 20))
```

**释义：**

1. **核心价差** ：计算 `vwap - close` ，即当日 **成交量加权平均价(VWAP)**  与 **收盘价(Close)**  的差值。
   - VWAP代表了市场全天的平均成交成本。
   - 收盘价是市场在交易时段结束时的最终定价。
   - **逻辑** ：如果 `VWAP - Close > 0` ，意味着收盘价 **低于** 全天平均成交成本。这可能表明在交易日尾盘出现 **卖压** ，股价被压制在均线之下。反之，若差值为负，则表明尾盘有 **买盘推动** ，收盘价高于全天均价。
2. **平滑处理** ：对这个价差序列取 **5日简单移动平均** 。
   - `sma(vwap - close, 5)`  的作用是平滑单日的噪音，形成一个短期趋势。连续多日收盘价低于VWAP，会使得这个均值维持正值且可能扩大。
3. **标准化（波动调整）** ：将上述5日均值除以收盘价的 **20日标准差** 。
   - `stddev(close, 20)`  衡量股票近20日的价格波动率。
   - **目的** ：消除不同股票波动性差异的影响。一个波动性很大的股票，其 `vwap-close` 的绝对值天然可能更大。除以波动率后，因子值更侧重于“异常程度”而非绝对大小，使不同股票间更具可比性。
4. **横截面排序** ：最后对全市场股票在截面上进行 `rank` 排名。
   - 将标准化后的值转化为在全市场股票中的百分位排名（通常归一化到0-1之间）。
   - **目的** ：这是一个纯粹的Alpha因子构造思想，旨在产生一个与未来收益率在 **横截面上** 具有单调性的预测指标。我们并不关心因子值的绝对大小，只关心它在所有股票中的相对位置。

### **核心逻辑与金融直觉**

该因子试图捕捉的是  **“尾盘弱势”或“价格压抑”的持续性效应** 。

1. **短期压力信号** ： `VWAP - Close` 为正，是 **单日看跌信号** 。它反映了当天从开盘到收盘，买盘力量未能将价格维持在全天平均成本之上，尤其是在收盘这个关键时点被压低，可能意味着机构调仓、大户出货或市场情绪转弱。
2. **趋势确认** ：通过5日移动平均，因子寻找的是 **连续数日出现这种压抑状态** 的股票。单日现象可能是噪音，但连续几日出现，则暗示卖方力量持续存在且可能未完全释放。
3. **风险调整** ：除以波动率是关键一步。在低波动股票中，一个较小的 `vwap-close` 差值可能已经算显著；而在高波动股票中，同样的差值可能只是日常噪音。这一步确保了因子度量的是 **经过风险调整后的异常压抑程度** 。
4. **反向交易假设** ：该因子的典型预期是，这种持续的非正常价格压抑（收盘价持续低于平均成本）是 **不可持续的** 。随着压抑力量的衰竭或被市场发现，股价存在 **向均值回复（反弹）**  的可能。因此，该因子通常被认为是一个 **短期的、反转类因子** 。
   - **注意** ：在某些市场环境下（如强烈的下跌趋势中），它也可能成为动量因子，即“弱势股持续弱势”。这需要通过历史回测来确定其在当前市场中的相位。
