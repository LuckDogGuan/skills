概述：量化投资(“Quant”)是一门结合了金融工程、计算机科学、数学、统计学等学科的交叉学科。在过去的几十年里，Quant已经成为主流投资方法之一，并经历了三个世代：Quant 1.0，通过数学建模来发现市场中定价错误的资产；Quant 2.0，将量化研究从小型“策略工坊”进化到大型“Alpha工厂”；Quant 3.0，应用深度学习技术发现复杂的非线性定价规则。尽管深度学习在预测方面有优势，但它依赖于极其庞大的数据量和对“黑箱”神经网络模型的劳动密集型调优。为了解决这些限制，在这篇文章中，作者介绍了Quant 4.0，并为下一代Quant提供了一个工程视角。首先，自动化的人工智能(“AI”)将量化从传统的手工建模转变为最先进的自动化建模，实践了“算法产生算法，模型构建模型，最终AI创造AI”的理念。其次，可解释的AI开发了新的技术，以更好地理解和解释由机器学习做出的投资决策，并解释复杂和隐藏的风险暴露。第三，知识驱动的AI是对深度学习等数据驱动的AI的补充，它将先验知识纳入建模，以改善投资决策，特别是量化价值投资。此外，作者还讨论了如何构建一个实践Quant 4.0概念的系统。最后，作者提出了量化技术的十个具有挑战性的研究问题，并讨论了潜在的解决方案、研究方向和未来趋势。

2. 一个公式

文中为了说明“因子挖掘”的概念，举了公式(5)这个例子，

```
factor=-ts_corr(rank(close),rank(volume),50)
```

其逻辑十分清晰。某支股票在之前的50个交易日内，如果

```
(a) 成交量上涨，(a.1) 同时价格上涨，说明涨到头了，我们就做空 (正相关)，(a.2) 同时价格下跌，说明跌到头了，我们就做多 (负相关)，(b) 成交量下跌，(b.1) 同时价格上涨，说明涨势可持续，我们就做多 (负相关)，(b.2) 同时价格下跌，说明跌势可持续，我们就做空 (正相关)。
```

特别是在return时序的开始几个点，直接调用pct_change，会生产80倍、甚至inf这样的return，将这样带有异常值的return时序直接喂入pd.corr函数，甚至会返回nan。
