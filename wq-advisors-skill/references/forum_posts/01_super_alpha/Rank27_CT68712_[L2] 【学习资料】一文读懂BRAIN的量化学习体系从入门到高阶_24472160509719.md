## 帖子正文

今天向大家介绍BRAIN的量化研究学习体系，这是一个专为不同阶段的量化爱好者和研究者设计的全面学习路径。无论你是量化领域的新手，还是寻求进一步提升的高阶研究者，BRAIN的学习体系都能为你提供所需的知识和工具。

> ```
> 1.入门阶段：
> ```

在这一阶段，我们的目标是帮助你理解什么是alpha，如何掌握运算符和数据集的使用，以及如何理解不同的回测设置和回测结果的各项指标。通过修改alpha示例，你将能够创建出满足提交标准的alpha，并通过累积challenge积分达到金牌水平。

**学习资源：**

- 在线直播课《零基础学量化》共四节（滚动推出请关注邮件通知）

- Day1: 开营第一课：量化行业初探及基本概念
- Day2: 使用快速表达式写出第一个量化策略
- Day3: 中性化与分组：以经典的基本面策略为例
- Day4: 增加策略平稳与稳健性：以经典的量价策略为例

- 自学路径图（入门篇） [【学习资料】BRAIN平台自学路径图推荐 – WorldQuant BRAIN]([L2] 【学习资料】BRAIN平台自学路径图推荐Pinned_22863075241623.md)

> ```
> 2.进阶阶段：
> ```

进阶阶段是从alpha示例到复现论文的过程。到了这一阶段，你已经掌握了基本知识，并准备进入alpha research的模仿到复现阶段。你将学会快速获取论文中主要的alpha idea的能力和技巧，使用data explorer找到论文中的数据集，并通过计算复现或近似复现论文的alpha ideas来形成信号，甚至可以提交的alpha。此时，用户往往已经成为我们的研究顾问。

**学习资源：**

- 在线直播课《带你读论文》共四节（滚动推出，请关注邮件通知）

- Day1: 抽丝剥茧，从论文中厘清Alpha必要要素，学习掌握如何从一篇论文中根据checklist快速提取重要信息
- Day2:  创作与提升，回测并提升Alpha Performance通过案例教学，掌握如何将checklist中的信息完成回测并提升
- Day3:  高效泛化，从一个Alpha表达式到多个Alpha表达式的拓展学习如何举一反三，从一篇论文内泛化成多个alpha ideas
- Day4:  Alpha Idea 分享会，1+1 大于2，你将有机会听到包括我们顾问在内分享的思维和灵感的碰撞

- 自学路径图（进阶篇）TBA

> ```
> 3.高阶阶段：
> ```

高阶阶段的用户仅限顾问权限，已经具备了通过论文复现alpha idea形成信号的能力。在这一阶段，学习目标是利用平台的API和海量数据构建自己的回测框架，泛化alpha idea模板，实现多数据集、多市场的大规模回测，从而提升研究效率和质量。此外，在提交超过50个alpha之后，用户可以开始解锁alpha组合的功能，构建super alphas。

**学习资源：**

- 在线直播课《英才候选人计划》/ 线下挑战营共五节

- 先导课：BRAIN API功能介绍详解，通过API可以不间断的完成1000次 simulation
- Class 1 :  Human + Machine 研究方法框架,从具体到抽象，通过论文，找到自己的template，并找到1个alpha
- Class 2：数据选择与Machine 研究框架完善，迭代Machine框架提升效率，通过更多的template找到4个alpha
- Class 3：基于遗传基因算法的路径升级与迭代，学习新的GA算法，提升数据与表达式搜索效率
- Class 4：提升Alpha的Value与“overfitting”的防范指南，Alpha质量提升与常见误区避雷

- 自学路径图（高阶篇）TBA
