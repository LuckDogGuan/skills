The system is well-structured and level-based. You can alternate between  **self SuperAlpha**  and  **non-self SuperAlpha**  submissions (once access is unlocked).

- **Gold Level** : Can create SuperAlphas using only their own Alphas
- **Expert Level** : Can use Alphas from their university pool
- **Master Level** : Can access Alphas from their country pool

### 评论 #5 (作者: JO96892, 时间: 1个月前)

For each SuperAlpha created using only the self pool, you earn a quota to create another SuperAlpha from other pools depending on your Genius level.

***Below is a Breakdown***

***Expert Level Consultants***

- Use both self pool and 'university' Alpha pool
- For each SuperAlpha created using only the self pool, you earn a quota to create another SuperAlpha from the university pool

***Master Level Consultants***

- Use both self pool and Alpha pool of their ‘country’
- For each SuperAlpha created using only the self pool, you earn a quota to create another SuperAlpha from the country pool

***Grandmaster Level Consultants***

- Use both self pool and the entire Alpha pool
- For each SuperAlpha created using only the self pool, you earn a quota to create another SuperAlpha from the entire pool
