资格问题：

提交的要求是什么？

参赛者必须在比赛期间在美国、欧元区、亚洲和大中华区至少提交 10 个 SuperAlphas 才能获得津贴。

SuperAlphas 必须使用链接到同一父 Alpha ID 的子 Alpha 构建。SuperAlphas 需要将父 Alpha ID 作为标签。

参赛者需要提交他们在比赛期间的研究过程所反映的程序代码（Jupyter Notebook）。

所有提交必须遵循提供的模板和指南。在提交 Jupyter Notebook 时，确保不包含您的 BRAIN API 或 LLM API 凭证。

竞赛流程是什么？

参赛者将使用他们现有的提交（称为父 Alpha）作为输入，生成新的 Alpha，使用他们选择的 LLM。这些新的 Alpha（称为子 Alpha）只能是 ATOM 或 PowerPool。然后，这些新的子 Alpha 将被用来创建一个 SuperAlpha，这将决定他们在排行榜上的位置。子 Alpha 需要使用 LLM 创建，这里提供了一个相同的模板。

什么是AI Alpha竞赛？

AI Alphas 竞赛是一个仅限个人参加的竞赛，参赛者使用 Python、外部 LLMs 和 BRAIN API 创建 Alphas 和 SuperAlphas。该竞赛鼓励利用 LLMs 进行基于机器的研究，以生成预测模型。

比赛期间提交配额是？

在 AIAC2025 期间，SuperAlpha 的提交配额仍然是 1。

由于需要许多子 Alpha 来构建这些 SuperAlphas，我们在比赛中放宽了 Alpha 的提交配额。

前四个 Alpha 提交将计入您的补偿和常规研究指标，第五个提交及以后，您的 Alphas 将仅计入 AIAC。AIAC 子 Alpha 没有每日提交配额。

允许团队合作吗？

仅个人参赛

我可以用任何模型来提交吗？

可以用任何模型，但是别改call_llm模版函数结构

能用比赛前创建的Alpha吗？

不行，仅比赛期间提交的Alpha和Sa能参与评分

如何报名？

需要手动进行，可在比赛标签页内完成

津贴如何？

1:5000  2:3500  3:2500  4-10:1200  11-25:700

至少提交十个才有资格

比赛时间？

持续六周，排行榜定期更新，最终排名将在程序评估后公布

谁有资格？

所有顾问和有条件顾问

自己的尝试和思考：

目前已经通过官方提供的模板和库信息进行了相关配置，官方给的代码还是比较全面的，ace库里丰富的内容不管是这次比赛还是往后都会用的上，有相当高的参考意义，还有一个计算pnl和统计信息的脚本文件，能用在这里也能用在以后跑sa上。

目前也只是提供了库和前两步工作流，也就是调用模型对alpha进行描述，然后生成新的表达式。根据我的实际体验下来，生成的表达式是不能直接用的，模型会使用你所没有的操作符，这块需要避雷一下，以上就是我的初步尝试。

从中午刚开赛时对于调用API花费问题有所心疼，到傍晚拼尽全力找免费api，再到晚上成功跑通了前两步，已经走成功了三步，感觉后续还有很多需要攻克的地方，欢迎大家在下方留言，一起讨论。

祝各位早日榜上有名，津贴拿满。
