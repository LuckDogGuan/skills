### 评论 #4 (作者: NS94943, 时间: 1年前)

Hi,

In addition to what others have suggested, you can also quite easily do that using SuperAlpha selection expression like this for example :

own * (in(datasets, "pv1") && os_start_date> " 2022-01-01")

When you run it, you will get a list of your alphas which have used the selected dataset ("pv1" in this case), which you can open in a new tab. The selection expression is more powerful than the UI.

If you want to list pyramid alphas, use something like  in(datacategories,"analyst") in the expression.

Cheers!
