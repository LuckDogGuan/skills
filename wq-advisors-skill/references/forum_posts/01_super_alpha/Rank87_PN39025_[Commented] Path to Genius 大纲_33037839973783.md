4. Osmosis Rank

- 显著影响Daily Payment**
- Combined Osmosis Performance**有助于Genius Level的晋级

4.1 Osmosis Points标准

- 全部类别的Alphas, 包含RA, ATOM, PowerPool, SA, etc.
- 至少3个Scope, 每个Scope下**至少**分配10个Alphas,并且要**正好**分配100,000。

4.2 Osmosis Points的分配方法

- 在Alpha页面的Settings中分配，
- 也可以使用ACE API library的set_alpha_properties函数进行分配，输入小于100,000的点数分配或者None来清除
- 分配后点数会实时更新在Genius页面中，可以切换到Genius页面进行确认

- 更多内容请参考： [https://platform.worldquantbrain.com/learn/documentation/consultant-information/osmosis-allocation-guide-consultants](https://platform.worldquantbrain.com/learn/documentation/consultant-information/osmosis-allocation-guide-consultants)

### 评论 #23 (作者: ZH34713, 时间: 16天前)

Path to Genius 大纲 1.5 Product Correlation 部分提到先本地做 self，后做 pc。这里补充一个最近的平台更新，对 PC 优化策略有直接影响。

根据 2026-05-22 平台公告， `ts_backfill`  和  `group_backfill`  两个算子现在不再计入 Genius 的平均算子数、总算子数，也不计入 Power Pool Alphas 的算子数统计。这意味着在优化产品相关性时，可以更自由地使用这两个算子来提升 alpha 覆盖率（coverage），而不用担心拉高算子数影响 Genius 排名。

具体可以参考官方公告中的帖子： [How to improve Alpha coverage with ts_backfill and group_backfill]([L2] [BRAIN TIPS] How can you improve Alpha coverage with ts_backfill and group_backfill置顶的_40576075554327.md)
