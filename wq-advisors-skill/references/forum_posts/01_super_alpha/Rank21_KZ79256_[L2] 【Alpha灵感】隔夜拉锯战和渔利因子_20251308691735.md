## 帖子正文

**灵感来源：**

广发证券 《隔夜“拉锯战”和渔利因子》

**因子描述：**

**主导隔夜交易的投资者和主导盘中交易的投资者可能是存在显著差异的。** 一般而言，相对专业的机构投资者和职业投资者倾向于在盘中进行交易；而隔夜交易者中，短线交易的投资者占比较高。隔夜交易和盘中交易的有效性也会存在显著的差异。

在一些新的研究中表明，许多股票和投资组合在每日交易周期中都会出现一些特定的模式。 **两种截然不同的投资者分别主导隔夜交易和盘中交易，而这两**  **类投资者不同的交易逻辑可能会打起“拉锯战”。**

在进行价格矫正的过程中，机构投资者可能会出现过度矫正的情况，即将隔夜的一部分有效交易误认为是由于噪音产生的错误定价。 **过度矫正会使股票定**  **价出现偏误，在未来有回归的趋势。**

**因子构建：**

研报提出了NR、PR、ABNR、ABPR四个因子

```
RET_OC = 盘中涨跌幅RET_CO = 隔夜涨跌幅NR = 当月出现反向盘中逆转的次数/当月实际交易天数PR = 当月出现正向盘中逆转的次数/当月实际交易天数ABNR = 当月NR/过去n个月NR的均值ABNR = 当月PR/过去n个月PR的均值
```

其中反向盘中逆转是指RET_OC<0，RET_CO>0，正向盘中逆转是指RET_OC>0，RET_CO<0

**Brain实现**

```
ret_oc = close/open-1;ret_co = open/ts_delay(close,1)-1;NR = ts_sum(ret_co>0&&ret_oc<0?1:0,66)/66;PR = ts_sum(ret_co<0&&ret_oc>0?1:0,66)/66;group = bucket(rank(cap),range='0,1,0.1');group_neutralize(rank(NR)+rank(-PR),group)
```

我等权重结合了NR（正向因子）和PR（负向因子）两个因子，在CHN Top3000上测试，原文在一个月时间窗口（22天）上构建因子，而我采用三个月的事件窗口（66天），发现因子表现更好，同时对冲了市值风险和行业风险（subindustry），表现进一步提升，最终可以提交！

隔夜交易：隔夜交易通常指的是在一天的交易结束后（即收盘后）至下一个交易日开始前的时间段内，投资者持有的某种资产或金融工具的交易。这种交易方式允许投资者在交易日结束后仍保留头寸，直至下一个交易日。隔夜交易中，短线交易的投资者占比较高。这些投资者通常利用市场波动和价格差异来谋取短期收益。

盘中交易：盘中交易指的是在股票交易时间段内（即开盘至收盘）进行的交易。这是金融市场中最常见的交易方式，也是大多数投资者所熟悉的交易方式。相对专业的机构投资者和职业投资者更倾向于在盘中进行交易。这些投资者通常拥有更丰富的市场知识和分析能力，能够更准确地把握市场走势和交易时机。

因子构建的逻辑旨在捕捉股票在每日交易周期中特定模式的表现，特别是隔夜交易与盘中交易之间的价格动态及其逆转现象。这些因子（NR、PR、ABNR、ABPR）旨在帮助投资者理解并分析由不同投资者群体（隔夜交易者和盘中交易者）主导的市场行为及其可能带来的价格矫正过度现象。

1. 盘中涨跌幅（RET_OC）：
   表示股票在盘中交易时段（即开盘至收盘）的价格变动百分比。

2. 隔夜涨跌幅（RET_CO）：
   表示股票在隔夜时段（即收盘至次日开盘）的价格变动百分比。

3. 反向盘中逆转次数（NR）：
   定义为当月中，盘中涨跌幅为负（RET_OC < 0）且隔夜涨跌幅为正（RET_CO > 0）的情况出现的次数。
   这表明股票在隔夜时段上涨，但在盘中时段下跌，即出现了“反向逆转”。

4. 正向盘中逆转次数（PR）：
   定义为当月中，盘中涨跌幅为正（RET_OC > 0）且隔夜涨跌幅为负（RET_CO < 0）的情况出现的次数。
   这表明股票在隔夜时段下跌，但在盘中时段上涨，即出现了“正向逆转”。

5. 平均反向盘中逆转率（ABNR）：
   定义为当月NR值除以过去n个月NR值的均值。
   旨在衡量当前月份反向逆转频率相对于历史水平的异常程度。

6. 平均正向盘中逆转率（ABPR）：
   定义为当月PR值除以过去n个月PR值的均值。
   旨在衡量当前月份正向逆转频率相对于历史水平的异常程度。
