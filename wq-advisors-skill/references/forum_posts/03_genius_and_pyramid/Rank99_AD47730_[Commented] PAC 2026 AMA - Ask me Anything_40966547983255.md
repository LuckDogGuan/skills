### 评论 #26 (作者: AG20578, 时间: 9天前)

Hi  [IK41094](/hc/en-us/profiles/32094182615703-IK41094) ,  [MM27048](/hc/en-us/profiles/37183389925655-MM27048)

> How to solve this error - Pure Power Pool submission does not match any Power Pool Theme.

Python Alphas are not eligible for Power Pool.

If you want to submit Fast Expression alpha that does not pass regular tests - you need to follow Power Pool Theme. Current Power Pool theme been launched 2 days ago, you can find an announcement:

[📢 Launching a new "GLB/D1 Dataset Power Pool June`26" theme](https://platform.worldquantbrain.com/messages/announcements)

[Power Pool](https://platform.worldquantbrain.com/learn/documentation/consultant-information/getting-started-power-pool-alphas)  Alpha should be on GLB region and delay "1". You can find the list of available datasets on  [Data page](https://platform.worldquantbrain.com/data/data-sets?delay=1&instrumentType=EQUITY&limit=20&offset=0&region=GLB&theme=true&universe=TOP3000)  with Theme checkbox.
Alphas that match the theme are eligible for Power Pool Thematic competition.
