## 帖子正文

Is there a way to view the actual operators that I have used on WorldQuant Brain? The Genius tab only shows the count. A feature to see them would greatly help in expanding operator usage.

### 评论 #1 (作者: LN92324, 时间: 1年前)

Hi. A manual way is you can go to the Submitted section, then use the Filter to filter the alphas in this quarter and then copy the alphas you have submitted to a file. Then you can manually review the operators in the learn/operators section to analyze which operators you have used and which operators you have not used.

### 评论 #6 (作者: DK20528, 时间: 1年前)

One manual approach is to go to the  *Submitted*  section and apply a filter to view only the alphas submitted this quarter. You can then copy them into a separate file. From there, you can review each alpha and cross-check the operators used by referencing the  *learn/operators*  page — this helps identify which operators you've used and which ones you haven't explored yet.

### 评论 #7 (作者: LB76673, 时间: 9个月前)

Improving the Combined score requires focusing on both quality and diversity. Submitting around 10–20 strong alphas each month, spread across regions and neutralizations, is usually more effective than submitting too few or too many. The real impact comes from keeping correlation low, maintaining OS strength, and using diversity to balance weaknesses so every alpha adds meaningful value.
