# How to Fix Simulation Issues Caused by Full Cache

- **链接**: How to Fix Simulation Issues Caused by Full Cache.md

## 帖子正文

Hi everyone,If you open multiple tabs for simulation, the cache may become full. As a result, when you click "Simulate," it might not display properly. To resolve this issue, you can try cloning the simulation to another tab to check if it is processing.Alternatively, you can clear the session and local storage:PressF12to open the developer tools.Navigate to theApplicationtab.Delete the session storage and local storage data.After doing this, you should be able to click "Simulate" and proceed normally.Hope this guide helps!
