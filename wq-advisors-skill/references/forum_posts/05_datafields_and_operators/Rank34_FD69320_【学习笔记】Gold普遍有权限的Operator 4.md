从左到右：今天，昨天，前天...d = 6result = ts_arg_min(x, d)  # 返回 1，因为最小值2出现在昨天（偏移量为1）使用示例：# 判断股票是否处于20天低点is_recent_low = ts_arg_min(close, 20) == 0# 如果最低点出现在较远的过去，可能表示上升趋势uptrend = ts_arg_min(close, 10) > 5# 分析最低点出现的频率low_point_frequency = ts_arg_min(close, 30)# 结合最高点和最低点的位置来判断趋势trend = ts_arg_max(close, 20) - ts_arg_min(close, 20)34. ts_mean功能说明：计算过去 d 天的平均值。它是一个时间序列操作符，用于计算移动平均。参数说明：- x: 输入的时间序列数据- d: 回溯的天数（lookback period）使用示例：# 判断多空趋势is_uptrend = close > ts_mean(close, 20)  # 价格在20日均线之上# 计算价格相对均线的偏离程度price_momentum = close / ts_mean(close, 10) - 1# 短期均线穿越长期均线golden_cross = ts_mean(close, 5) > ts_mean(close, 20)# 计算价格对均值的偏离volatility = close - ts_mean(close, 10)35. ts_backfill功能说明：用于填充缺失值（NaN）的时间序列操作符。它会用历史数据中最近的有效值来替换当前的 NaN 值。参数说明：- x: 输入的时间序列数据- lookback: 回溯的天数（默认为d）- k: 选择第k个非NaN值（默认为1，即最近的有效值）- ignore: 指定要忽略的值（默认为"NAN"）计算过程：# 假设某股票最近6天的数据如下：Day 0 (今天):   NaNDay -1 (昨天):  NaNDay -2:         95Day -3:         NaNDay -4:         90Day -5:         85k=2时，使用ts_backfill的结果result = ts_backfill(x, lookback=6, k=2)# 结果：Day 0:          90     # 用第二个有效值90填充Day -1:         90     # 用第二个有效值90填充Day -2:         95     # 原始值保持不变Day -3:         85     # 用第二个有效值85填充Day -4:         90     # 原始值保持不变Day -5:         85     # 原始值保持不变使用示例# 股票停牌期间用最近的交易价格填充clean_price = ts_backfill(close, 252)# 财务数据发布间隔期间用最新的数据填充clean_profit = ts_backfill(net_profit, 252)36. greater_than_equal功能说明：相当于>=, 是一个逻辑比较操作符，用于比较两个输入值。如果第一个输入大于或等于第二个输入，返回 true (1)；否则返回 false (0)。参数说明：- x: 第一个输入值- y: 第二个输入值37. less_than_equal功能说明：相当于<=, 是一个逻辑比较操作符，用于比较两个输入值。如果第一个输入小于或等于第二个输入，返回 true (1)；否则返回 false (0)。参数说明：- x: 第一个输入值- y: 第二个输入值38. not_equal功能说明：相当于!=, 是一个逻辑比较操作符，用于比较两个输入值是否不相等。如果两个输入值不相等，返回 true (1)；如果相等，返回 false (0)。参数说明：- x: 第一个输入值- y: 第二个输入值39. equal功能说明：相当于 \==, 是一个逻辑比较操作符，用于比较两个输入值是否相等。如果两个输入值相等，返回 true (1)；如果相等，返回 false (0)。参数说明：- x: 第一个输入值- y: 第二个输入值40. greater功能说明：相当于 >, 是一个逻辑比较操作符，用于比较两个输入值。如果第一个输入严格大于第二个输入，返回 true (1)；否则返回 false (0)。参数说明：- x: 第一个输入值- y: 第二个输入值41. is_nan功能说明：是一个逻辑判断操作符，用于检测输入值是否为 NaN (Not a Number)。如果输入是 NaN，返回 1 (true)；如果输入不是 NaN，返回 0 (false)。参数说明：- x: 输入值，可以是任何数值类型42. densify功能说明：用于将具有多个桶(buckets)的分组字段转换为更少的可用桶，以提高分组字段的计算效率。它会将稀疏的分组值映射到一个连续的、密集的索引范围内。参数说明：- x: 输入的分组字段，通常是整数值计算过程# 原始分组值：industry_codes = {0, 1, 2, 99}  #
