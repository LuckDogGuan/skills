## 21. ts_av_diff

***功能说明*** ：计算当前值与过去d天平均值的差值，同时会特殊处理NaN值（在计算平均值时忽略NaN）。

***计算公式：***

```
ts_av_diff(x, d) = x - ts_mean(x, d) # 其中ts_mean在计算时会忽略NaN值
```

***参数说明：***

- - x: 输入数据
- - d: 回看天数

***使用示例：***

```
# 计算价格相对于均线的偏离price_deviation = ts_av_diff(close, 20)  # 相对20日均线偏离# 计算成交量异常volume_anomaly = ts_av_diff(volume, 5)   # 相对5日均量偏离# 计算收益率异常returns_anomaly = ts_av_diff(returns, 10) # 相对10日平均收益偏离
```

## 22. group_backfill

***功能说明：*** 当某个数据点为NaN时，使用同组内其他资产过去d天的数据计算经过winsorize处理的均值来填充。

***参数说明：***

- - x: 需要填充的数据
- - group: 分组依据（如行业、部门等）
- - d: 回看天数
- - std: winsorize的标准差倍数（默认4.0）

***工作原理：***

1. 识别NaN值
2. 收集同组资产过去d天的非NaN值
3. 计算这些值的均值和标准差
4. 进行winsorize处理（截断异常值）
5. 用处理后的均值填充NaN

***计算过程*** 

```
# 假设有3只股票(i1, i2, i3)在同一个组，过去4天的数据：i1 = [4,  2, 5, 5]    # 最新数据在最左边i2 = [7,  NaN, 2, 9]i3 = [NaN, -4, 2, NaN]# 处理步骤：1. 找出需要填充的NaN：i3的最新值2. 收集同组内所有非NaN值：[4, 2, 5, 5, 7, 2, 9, -4, 2]3. 计算统计量：mean = 3.56stddev = 3.71截断范围 = [mean - 4*stddev, mean + 4*stddev]        = [3.56 - 4*3.71, 3.56 + 4*3.71]        = [-11.28, 18.40]        4. 由于所有值都在截断范围内，不需要winsorize处理winsorized_mean = 3.565. 虽然i3确实有两个NaN值（最新时点和最早时点），但group_backfill操作符在这个例子中只关注最新时点的值。其他时点的NaN值（比如i2第二个时点的NaN和i3最后时点的NaN）在这个例子中并没有被处理。
```

```
完整的填充结果应该是：i1 = [4,  2, 5, 5]     # 保持不变i2 = [7,  NaN, 2, 9]   # 保持不变i3 = [3.56, -4, 2, NaN] # 只有最新时点的NaN被填充为3.56
```

***使用示例：***

```
# 处理财务数据缺失clean_pe = group_backfill(pe_ratio, industry, 60)# 处理成交量缺失clean_volume = group_backfill(volume, sector, 20)# 小市值股票数据质量较差时的处理clean_data = group_backfill(    raw_data,    market_cap_group,    30,    std=5.0  # 使用较大的std以保留更多信息)
```

## 23. scale

***功能说明*** ：将输入数据缩放到指定的仓位规模（booksize）。可以分别为多头和空头设置不同的缩放比例。

***参数说明：***

- - x: 输入数据
- - scale: 整体缩放比例（默认为1）
- - longscale: 多头仓位缩放比例（默认为1）
- - shortscale: 空头仓位缩放比例（默认为1）

***计算原理：***

1. - 首先将所有数据的绝对值之和缩放为1
2. - 然后根据scale参数进行整体缩放
3. - 最后根据longscale和shortscale分别对多空头部分进行缩放

***计算过程：***

```
x = [2, -1, 4, -2, 1]  # 5个股票的信号# 计算绝对值之和abs_sum = |2| + |-1| + |4| + |-2| + |1| = 10# 归一化normalized = [    2/10  = 0.2,    -1/10 = -0.1,    4/10  = 0.4,    -2/10 = -0.2,    1/10  = 0.1]# scale(x, scale=1)result = [0.2, -0.1, 0.4, -0.2, 0.1]# scale(x, scale=2)result = [0.4, -0.2, 0.8, -0.4, 0.2]# scale(x, longscale=1.5, shortscale=0.5)result = [    0.2 * 1.5  = 0.3,    # 多头    -0.1 * 0.5 = -0.05,  # 空头    0.4 * 1.5  = 0.6,    # 多头    -0.2 * 0.5 = -0.1,   # 空头    0.1 * 1.5  = 0.15    # 多头]
```

***使用示例：***

```
# 保守策略scale(signal, scale=0.5)  # 半仓# 激进策略scale(signal, longscale=1.5, shortscale=1.5)  # 1.5倍杠杆# 非对称策略scale(signal, longscale=1.2, shortscale=0.8)  # 多头偏向
```

## 24. ts_regression

***功能说明*** ：在d天窗口内进行最小二乘法(OLS)线性回归，可以返回多种回归相关的统计量。

***参数说明：***

- - y: 因变量（被解释变量）
- - x: 自变量（解释变量）
- - d: 回看天数
- - lag: 滞后期数（默认0）
- - rettype: 返回值类型（默认0），决定返回哪种统计量

**rettype参数对应的返回值：**

```
| rettype | 返回值        | 说明    | 含义               || ------- | ---------- | ----- | ---------------- || 0       | Error Term | 残差项   |  实际值与预测值的差异      || 1       | α (Alpha)  | y轴截距  | 表示X=0时Y的预测值      || 2       | β (Beta)   | 斜率系数  | 表示X变动一个单位时Y的变动量  || 3       | y-estimate | 预测值   | 模型预测值            || 4       | SSE        | 误差平方和 | 反映模型未能解释的变异      || 5       | SST        | 总平方和  | 反映Y的总变异          || 6       | R-Square   | 判定系数  | 反映模型的解释能力（0-1之间） || 7       | MSE        | 均方误差  | 反映预测的平均精度        || 8       | SE(β)      | β的标准误 | 用于检验β的显著性        || 9       | SE(α)      | α的标准误 | 用于检验α的显著性        |
```

***计算原理***

```
# 假设我们有5天的数据：# 股票收益率(Y)和市场收益率(X)的数据：x = [0.02, 0.03, -0.01, 0.04, 0.01]  # 市场收益率y = [0.01, 0.02, -0.02, 0.03, 0.01]  # 股票收益率
```

```
# 基础统计量：n = 5  # 样本数x_mean = (0.01 + 0.02 + (-0.02) + 0.03 + 0.01) / 5 = 0.01y_mean = (0.02 + 0.03 + (-0.01) + 0.04 + 0.01) / 5 = 0.01
```

**rettype=2**

```
# 计算β = Σ((X-X̄)(Y-Ȳ)) / Σ(X-X̄)²# (X-X̄)(Y-Ȳ)的计算：[( 0.01-0.01)(0.02-0.018) = 0 ( 0.02-0.01)(0.03-0.018) = 0.00012 (-0.02-0.01)(-0.01-0.018)= 0.00084 ( 0.03-0.01)(0.04-0.018) = 0.00044 ( 0.01-0.01)(0.01-0.018) = 0]Σ((X-X̄)(Y-Ȳ)) = 0.0014# (X-X̄)²的计算：Σ(X-X̄)² = 0.0022β = 0.0014/0.0022 = 0.636
```

**rettype=1**

```
α = y_mean - β * x_meanα = 0.018 - 0.636 * 0.01α = 0.0116
```

**rettype=3**

```
# Ŷ = α + βXpredictions = [    0.0116 + 0.636 * 0.01 = 0.0180    0.0116 + 0.636 * 0.02 = 0.0243    0.0116 + 0.636 * (-0.02) = -0.0011    0.0116 + 0.636 * 0.03 = 0.0307    0.0116 + 0.636 * 0.01 = 0.0180]
```

**rettype=0**

```
# ε = Y - Ŷerrors = [    0.02 - 0.0180 = 0.0020    0.03 - 0.0243 = 0.0057    -0.01 - (-0.0011) = -0.0089    0.04 - 0.0307 = 0.0093    0.01 - 0.0180 = -0.0080]
```

**rettype=4**

```
# sse = Σ(Y - Ŷ)²sse = 0.0020² + 0.0057² + (-0.0089)² + 0.0093² + (-0.0080)²    = 0.000251
```

**rettype=5**

```
# sst = Σ(Y - Ȳ)²sst = (0.02-0.018)² + (0.03-0.018)² + (-0.01-0.018)² +      (0.04-0.018)² + (0.01-0.018)²    = 0.00166
```

**rettype=6**

```
r² = 1 - (sse/sst)   = 1 - (0.000251/0.00166)   = 0.849 # 说明模型解释了84.9%的变异
```

**rettype=7**

```
mse = sse/(n-2)    = 0.000251/3    = 0.0000837
```

**rettype=8**

```
# se(β) = √(mse/Σ(X-X̄)²)se_β = √(0.0000837/0.0022) = 0.195
```

**rettype=9**

```
# se(α) = SE(β)√(Σx²/(n·Σ(X-X̄)²))se(α)= 0.195 * √0.1727 = 0.081
```

## 24. zscore

***功能说明*** ：zscore是一个统计工具，用于衡量数据点与平均值的偏离程度，以标准差为单位进行度量。

***计算公式：***  zscore = (x - mean(x)) / std(x) - 均值始终为0，标准差始终为1。

***含义：***

- - 正值：表示数据点高于平均值
- - 负值：表示数据点低于平均值
- - 绝对值：表示偏离平均值的标准差个数

***参数说明：***

- - x: 输入数据

***使用示例：***

```
# 标准化不同量级的因子normalized_factor = zscore(raw_factor)# 识别异常值（通常|zscore| > 3被视为异常）is_outlier = abs(zscore(values)) > 3# 极端值处理cleaned_signal = if_else(    abs(zscore(signal)) > 3,    sign(signal) * 3 * std(signal) + mean(signal),    signal)
```

## 26. quantile

***功能说明*** ：将输入数据转换成特定的分布形式，标准化数据到一个合理的范围内，以此减少异常值的影响。

***参数说明：***

- - x: 输入数据
- - driver: 分布类型

> - "gaussian": 高斯分布（默认）
> - "uniform": 均匀分布
> - "cauchy": 柯西分布

***计算过程：***

```
# 原始数据： [10.5, 25.8, 15.2, 42.3, 18.7]# 排序和归一化： [0.2, 0.8, 0.4, 1.0, 0.6]# uniform[0.2, 0.8, 0.4, 1.0, 0.6] -> [-0.6, 0.6, -0.2, 1.0, 0.2]
```

***uniform的特点***

1. - 结果近似线性分布
2. - 值的间隔更均匀
3. - 极端值的幅度相对较小
4. - 适合需要保持线性关系的场景

```
# gaussian[0.2, 0.8, 0.4, 1.0, 0.6] -> [-0.84, 0.84, -0.25, 1.28, 0.25]
```

***gaussian的特点***

1. - 结果呈现正态分布特征
2. - 中间值接近0
3. - 两端的值被拉得更开（有更大的正负值）
4. - 适合需要强调极端值差异的场景

```
# cauchy[0.2, 0.8, 0.4, 1.0, 0.6] -> [-1.38,  1.38, -0.32,  3.08,  0.32]
```

***cauchy的特点***

1. - 尾部更重（heavy-tailed）
2. - 极端值被拉得更开
3. - 中间值的变化相对温和

## 28. ts_delay

***功能说明*** ：返回d天前的x值。这是一个非常基础但很实用的时间序列操作符。

***參數说明：***

- - x: 输入数据
- - d: 回看天数

***使用示例：***

```
# 计算1日收益率returns = close / ts_delay(close, 1) - 1# 计算5日收益率returns_5d = close / ts_delay(close, 5) - 1# 简单动量策略momentum = close / ts_delay(close, 20) - 1  # 20日动量
```

***特点：***

- - 会产生NaN值（当没有足够的历史数据时）
- - 常用于构建时间序列特征
- - 是很多复杂指标的基础组件
