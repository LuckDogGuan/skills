1. ***常见的数据增强方法总结***

在基本面数据、分析师数据以及一些另类数据中，由于数据分布不均匀或有异常值等问题，会影响alpha的表现，例如：空缺值太多导致weight concentration；经济意义上看，这些alpha的“数值”在不同资产之间可能不具备可比性，例如：不同行业的EP无法直接比较，因为不同行业的估值逻辑是不一样的。因此，我们需要使用数据变换或者数据增强的方法赋能alpha构建。以下是我个人用到过或者听其他同学分享过的一些方法：

***1.1 利用分布本身的特性填充*** 
我们假设有一个分析师预测值（净利润增速），那么一般可能分析师喜欢预测具有成长性的企业，此时成熟性的企业可能覆盖率不全。常见的做法为，对每个行业内求该datafield的均值或中位数，对同一行业的空缺值进行填充，可能的代码如下：

```
A_robust=is_nan(A)==1?group_median(A,sector): AB_robust=is_nan(B)==1?group_mean(B,1,sector): B
```

**好处：**

1. 简单易懂，理念是同一行业的公司之间可比性较强、公司表现相近，因此即使值有空缺，使用均值和中位数填充，也可以得到一些抗干扰的预估

**坏处：**

1. 每个行业中的数据覆盖率决定了填充效果：

- 对于覆盖率本身就很低的datafield，可能会出现某个行业本身就没有这个datafield，假设A行业一定不会使用石油，而datafield是一个石油资产储备，此时可能填充不了；
- 还会出现过拟合的问题，比如某个行业只有A和B两个公司有值，而剩余有100个公司都是空值，即使填充，也不会给出合理的估计，甚至会导致填充后的数据在整个市场上的分布变得异常。

1. 每个行业中的数据分布决定了填充效果：如果行业内的datafield本身就分布不均匀，均值不会是一个很好的选择，中位数相对更鲁棒，但本质上可能并无法给出想要的估计

***1.2***  ***利用分布本身的特性进行变换***

在1.1中还有一个严重的问题是，对于某些数据如预期净利润（而非增长率），此时数据就会有单位，不可比。此时引入进一步做法，进行行业内zscore

```
A_robust=group_zscore(A,sector)
```

**好处：**

1. 简单易懂，消除了量纲（即行业整体水平带来的影响），在截面上更具可比性（类似alpha的中性化）

**坏处：**

1. 最受影响的可能还是数量的问题，过拟合。想象一个情况，每个行业只有三个公司，特别是当这里的“行业”是更细致的分类时，比如subindustry，可能会出现这个问题，此时zscore后大家都是0，无法合理的产生资产的权重。

***1.3 利用其他的数据进行预测***

这个我还没有深入尝试过，为大家提供一个思路，这是我在一个分享会上学习到的。当时这位同学说，利用一些常见的alpha对空缺值进行预测。比方讲，使用Barra风格因子对数据进行预测，经济依据是风格因子本身可以反应一些公司的共性特征（在短期的交易层面上可能是动量、波动率、流动性，长期的财务表现上可能是BP、盈利质量、成长性），通过截面拟合并预测的方式，从而填充空缺值。可能的代码如下：

```
A_robust=group_vector_proj(A, B, sector) # B为某个你认为很好的解释变量
```

**好处：**

1. 相比使用均值填充，这种方法更能凸显出被填充公司的差异性（均值填充会造成被填充的公司共用同一个值，还是不具备差异化）

**坏处：**

1. 因为还没有尝试过，我只能从直觉上说：这非常依赖于解释变量！regression终究还是regression，它本质上还是统计工具，虽然intuition很好，但是如果B和A的Pearson相关性不高，此时预测值应该非常不靠谱（有人提到Barra本身不能算是一个非常完美的模型）

***1.4 利用时序数据进行填充***

一种是直接使用ts_backfill算子进行填充，这种和截面均值填充一致的问题类似，虽然易懂，但你的值是静态的，尤其是当你不知道为什么产生空缺值的时候，可能会造成数据不具备可比性、不具备时效性等各种问题。另外一种则是时序预测，使用ts_regression调整rettype返回预测值即可，问题依然和截面预测类似，依然依赖于变量的选取。

***1.5 对数据强行转换***

这是本文想提出的重点方法。第一种是《基本面因子的重构》所提出的，非常创新地将财务比率的除法转换为回归，以下是简单地对减法、除法和回归进行总结：

1. 回归：一元回归表达式（先不考虑截距项）为：Y=aX+ε，ε为残差，期望为0
2. 减法：回归系数为1的线性回归。减法所得到的差，即为ε=Y-X
3. 除法：回归系数为1的双对数线性回归。此时的Y和X都是经过对数变换，Y=log(Y’), X=log(X’)，除法所得到的商，即为ε=Y-X=log(Y’)-log(X’)=log(Y’/X’)，对等式两边取exp，我们可以得到Y’/X’=exp(ε)=ε‘

既然如此，化用到财务比率上，比如EP（净利润除以总市值），其实暗含的假设是E和P的关系为1，即P改变1单位，E改变1单位。自然地，我们可以提问：真的是这样变化吗？不同公司间是不是应该变化的幅度不同？因此，既然回归系数可以等于1，我们也可以通过回归去动态的确定这个回归系数，比如通过截面回归，找到全市场的估值中枢。

**具体做法如下：** 1. 首先对数据进行增强（即采用log使数据尽可能正态）；2. 进行截面回归；3. 如果数据中有负数，为了可以log，进行sign(A)log(abs(A)+1)调整，+1是为了防止数值为0。以EPTTM为例，

```
A = mdl175_netprofitttm;B = mdl175_mktvalue;A_robust = sign(A)*log(abs(A)+1);B_robust = sign(B)*log(abs(B)+1);regression_neut(A_robust,B_robust)
```

但这样会出现一个问题，我们在回归中预期数据是服从正态的，但经过这样的调整，会出现双峰，可能对回归结果造成影响，因为数据有正有负；一种解决方法是按正负号分类，分别进行回归，但负样本比较少，可能会过拟合

**具体地我们给出template：**

1. 财务比率

```
A = mdl175_netprofitttm;B = mdl175_mktvalue'A/B
```

1. 对数回归

```
A = mdl175_netprofitttm;B = mdl175_mktvalue;A_robust = sign(A)*log(abs(A)+1);B_robust = sign(B)*log(abs(B)+1);regression_neut(A_robust,B_robust)
```

1. 高斯秩回归（注意，所有的没有解析解的数学公式，都可以使用近似公式代替，该公式来源wiki百科，此时误差为O(n^7)）

```
A = (rank(mdl175_netprofitttm)-0.5)*2;B = (rank(mdl175_mktvalue)-0.5)*2;pi = 3.1415926;z_A = sqrt(pi)/2*(A+pi/12*power(A, 3)+7*power(pi,2)/480*power(A,5));z_B = sqrt(pi)/2*(B+pi/12*power(B, 3)+7*power(pi,2)/480*power(B,5));regression_neut(z_A,z_B)
```

**2. 分享一个成熟的template**

最近我也在使用这个template进行alpha生成。我分享一些我对这个模板的一些应用，

1. **我将盈利类数据跳出来（比如other466中调出net inc这种，容易出信号），优先参与simulate**
2. **单独的fundamental字段往往效果不好，可以搭配一个盈利类字段，再搭配一个其余财务数据（这个财务数据不需要有很好的表现，只是对盈利类数据进行补充和优化，希望能达到1+1>2的效果）**
3. **优化论坛中已有的fundamental alpha的模板，只要它用到了财务指标！**

**基于此，我使用了 [【Alpha灵感】从布林带到估值异常因子 – WorldQuant BRAIN]([L2] 【Alpha灵感】从布林带到估值异常因子_20235808598423.md) 中的内容，具体的template如下，思路是先通过高斯秩进行数据增强，利用回归修正财务比率，套用到其他alpha上进行优化。**

```
A = (rank({data1})-0.5)*2;\B = (rank({data2})-0.5)*2;\C = (rank(cap)-0.5)*2;\A_robust = is_nan(A)==1?group_median(A,sector):A;\B_robust = is_nan(B)==1?group_median(B,sector):B;\pi = 3.1415926;\z_A = sqrt(pi)/2*(A_robust+pi/12*power(A_robust, 3)+7*power(pi,2)/480*power(A_robust,5));\z_B = sqrt(pi)/2*(B_robust+pi/12*power(B_robust, 3)+7*power(pi,2)/480*power(B_robust,5));\z_C = sqrt(pi)/2*(C+pi/12*power(C, 3)+7*power(pi,2)/480*power(C,5));\group_neutralize(ts_zscore(rank(regression_neut(z_A,z_C))+rank(regression_neut(z_B,z_C)),252),densify({group_field}))
```

整体上可探索性还比较高，比如cap也可以替换成其他数据引入新的自由度；另外，这种财务比率的修正是运算符号带来的问题，对估值逻辑上没有很强的变化，因此可以尝试在不同的region进行生成，最后是一个目前比较好的alpha绩效
