## 帖子正文

更新日志：

1. 使用uv统一管理虚拟环境，不需要再卸载重装特定版本的系统级Python

2. 在运行一开始/过程中主动申请提权，避免用户因没有使用管理员权限运行导致无法输入

3. 允许用户配置自己的模型，模型选择更多样

4. 可选自动构建基础知识库

---------------------------------------------以下为PC版本--------------------------------------------------

### 1. 下载安装包

```
通过网盘分享的文件：AI打工人链接: https://pan.baidu.com/s/19GghIxf1BwNdBf056eifgA?pwd=7rmy 提取码: 7rmy
```

### 2. 安装AI打工人

双击"BRAIN *工具一键安装* windows版本.exe" - 以管理员身份运行，选择(强烈建议新建)一个文件夹一键安装

### 3. 安装完成后配置打工人

a. 选择你用的大模型(Kimi/DeepSeek)或手动输入你的模型配置

b. 输入你的API Key

c. 选择你的模型

d. (重要！) 重启终端

e. 在新的终端里切换到你的打工人目录，输入claude，进入Claude Cli测试一下是否配置成功

### 2. 解压安装包

```
cd ~/Downloadsmkdir -p ~/AIWorker && unzip "AIWorker_MacLinux.zip" -d ~/AIWorker && cd ~/AIWorker
```

### 3. 安装AI打工人

```
# 如果说 Permission Deined 你需要切换一下所有权ls -ld AIWorker_MacLinuxsudo chown -R $USER:$USER AIWorker_MacLinux# 运行脚本zsh/bash install.sh# 如果无法执行脚本运行以下命令给脚本执行权限chmod +x install.sh
```

### 4. 安装完成后配置打工人
