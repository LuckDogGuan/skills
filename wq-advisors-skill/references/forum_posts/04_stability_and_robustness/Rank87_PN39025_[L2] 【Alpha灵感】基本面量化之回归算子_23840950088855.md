1. ***Quantamental的本质***

笔者在过去几个月中，阅读了[1]，[2]，[3]三本书，第一本是纯综述，可以作为量化投资在认知上的启蒙，它梳理了市场效率、行为金融和基本面分析领域的理论体系以及部分实证研究；第二第三本处于综述和实践之间，先是总结了在数十年间海外资产定价和基本面量化的前沿工作，方向包括盈利类因子、成长类因子、分析师、文本分析、股票间关联等热点策略，进一步探讨了这些策略在中国市场的可用性并给出实践性的建议。

**由于只是第一轮读，笔者只能简单总结：这三本书都在传达价值投资的核心是购买**  **“**  **相对廉价**  **”**  **和**  **“**  **优质**  **”**  **的股票。相对廉价，即站在同一时点（即一个截面），近似财务状况的公司中，某公司的股价较低（此处**  **“**  **相对廉价**  **”**  **是和**  **“**  **廉价**  **”**  **做对比，如果一个公司的股价极低、过于廉价，那也有可能是该公司正处于财务风险中）；**  **“**  **优质**  **”**  **则不用多说，在股价相近的一些票中，选择经营状况好、盈利能力强、安全性高的股票，这些状态都是可以反应股票的财务状况足够好、基本面足够强。**

1. ***为什么单纯的财务指标很难是真正的**  **alpha***

常见的财务静态结构（如衍生指标，流动比率、ROA等）、动态结构（如单季度净利润同比变化率），这些你不能说它完全没用，但是这些往往过于简单，这会导致即使它能赚钱，也不是market-neutral的。一个非常实时的例子是，CHN市场的单季度ROA，

1. 从21年之后，多空收益出现大幅回撤，而在此之前，18年至21年间，该alpha有稳定的收益
2. 如果我们再把时间放长，你更可以发现该alpha在14年至18年之间方向完全相反

这实际上说明这是一个beta。比如近期回撤的原因是随着机构抱团的瓦解，市场已经不再偏好大盘蓝筹风格的股票，而在过去的时间里投资者已经充分交易甚至过度交易了这些股票的盈利属性，这导致高盈利能力的公司股票预期已充分体现，估值过高，未来表现不佳【此处参考了东北证券的特质盈利因子的研报】。另外，也有实证表示大部分财务指标都不具备选股能力。

1. ***如何用回归算子刻画“廉价”和“优质”***

在吸收理论知识后，我转向阅读金融工程研报，参考在实务工作中如何构建基本面因子。在阅读大量资料后，我发现在截面选股中，regression是一个非常重要的构建方式，而我恰好又找到了两篇不同角度的regression，时序和截面。

- 截面regression

[4]中使用截面regression去重构财务衍生指标，试图找到比值中枢。研报主要想财务衍生指标的两个问题：1）除数效应，当分母取值为0时，可能会有异常值。虽然此处EPTTM除数是总市值，不会有分母趋近为0的情况，但也会有相对极小值的情况，可以想一个微小盘股票；2）定价绝对，长久以来，基本面量化的痛点是，不同公司之间的财务指标是否可比，通常做法为在选股前对其进行行业市值中性化，但这样并不能完全消除截面上的方差差异，从而导致与排序选股仍有错配。

- 时序regression

两个可行的思路是1. 超预期；2. 财务状况持续，这两个都可以用来刻画优质公司。

对于第一个思路，常见的因子是基于分析师数据的SUE，而 **[5]** 假设我们可以通过一个财务科目解释另一个财务科目，比如在从生产到销售，我们需要先付出制造商品的成本，才可以获得经营收益，那么我们可以尝试建立成本和收益之间的线性关系；又比如通过公司资产规模越大，收获的净利润可能越多；总之，我们可以对每支股票回望过去N日，建立财务科目之间的回归模型，最后取最后一期残差。在不同角度下（盈利、成本、安全性），该因子有正有负，但都可以判断公司在这个角度上是否超预期，例如，净利润和总资产进行时序回归，如果残差越大，说明公司盈利能力有明显增强、可能有特质盈利能力，超出自身历史表现。

我随机选取了一对基本面指标，发现有信号，因此可以通过该方法不断迭代。需要注意的是1）该方法自由度很大，除了财务科目外，还可以迭代财务衍生指标，据经验，有信号的因子有很多且相关性极低；2）在平台上对财务科目回归，如果设置两年500日，可能会有低估数据方差的风险，使得统计量估计有偏。

**总结：**

1. 基本面量化本质是找“相对廉价”和“优质”公司

2. regression在基本面因子构建中比较有效

3. 财报科目种类繁多，会导致simulate时自由度过大，可能有效率低的风险

4. 注意平台上财务数据的数据结构，尽量避免统计量的错误估计，建议使用last_diff_value（虽然我还没有尝试出来）

### 评论 #2 (作者: YW93864, 时间: 2 years ago)

Hi, there.

以下提供了alpha模板以及对今天的分享做了一些总结

1. 推荐想在基本面量化赛道做出alpha的同学，阅读Alphanomics——The Informational Underpinnings of Market Efficiency（中文版为阿尔法经济学），《基本面量化投资（第二版）》，《基本面量化投资——18种经典策略》入门
2. 截面回归模板

统计意义：处理除数效应以及修正数据在截面上的分布

经济意义：使财务比率在截面上具备可比性，找到“比值中枢”，刻画“相对廉价”，从而抛弃A除以B这样的“自身绝对定价方法”

代码说明：finance_var代表财务变量；sign提取了财务的符号（净利润可能为负，无法直接log）；log(abs+1)实际上进行了λ=0的boxcox变换，使得分布更倾向正态分布（使用标准化的数据进行回归，拟合效果会更好）

```
A = sign(finance_var)*log(abs(finance_var)+1));B = sign(finance_var)*log(abs(finance_var)+1));regression_neut(A,B)
```

3. 时序回归模板

3.1 财务指标A和B进行回归取最后一期残差

统计意义：寻找线性关系，取残差反应拟合效果

经济意义：以盈利-成本为例，一份耕耘一份收获，付出的成本越多，获取的利润越大，如果最近一期的残差过大过小说明公司盈利能力有显著变化，此处过大表示公司有盈利能力，因此alpha为momentum，这样找出业绩出彩的优质公司

代码说明：使用标准化的数据进行回归，拟合效果会更好，因此进行ts_zscore

```
ts_regression (ts_zscore(A,500), ts_zscore(B,500),500)
```

3.2 时序回归后，取回望周期内的标准差倒数

统计意义：残差标准差反应拟合效果

经济意义：3.1提供了方向性的alpha，而此处提供了衡量离散程度的alpha。如果该标准差越小，说明财务经营状况越稳定

```
1/ts_std_dev(ts_regression (ts_zscore(A,500), ts_zscore(B,500),500)，500)
```

3.3 时序回归后，进行残差标准化

统计意义：衡量当前值是否显著（例如假设alpha值为1.96，此时在单尾95%置信水平下显著）

经济意义：互补了3.1和3.2，我们希望找到财务水平稳定又能有突出表现的公司

```
residual = ts_regression (ts_zscore(A,500), ts_zscore(B,500),500);residual/ts_std_dev(residual ，500)
```

3.4 财务状况的时间动量效应

统计意义：衡量财务指标自身是否有线性关系

经济意义：我们期待过去财务状况好的公司，未来会更好，因此存在线性关系；同样可以取残差、残差标准差或者残差标准化

代码说明：此处以最后一期残差为例，回归的前者是财务指标，后者是timestep(500)，由于我们使用了500天的数据，因此设置步长为500

```
ts_regression (ts_zscore(A,500), timestep(500),500);
```

最后的注意和tips:

1. 后续可以尝试同时结合截面回归和时序回归。通过截面回归处理财务数据的分布，通过时序回归找出robust的signal

2. 缩小space。上述财务数据并没用限制，因此空间非常大；建议从盈利成长、成本、经营、安全性等重要维度先进行搜索，另外，可以先不考虑覆盖率较低的datafield

3. 可以用分析师数据代替财务数据，比如净利润

4. 基本面量化建议从经济逻辑出发，也不建议使用很复杂的公式，容易overfitting（因为数据点较少，很容易fit噪音）

5. 注意窗口设置时的500天，这实际上会导致回归时变量的方差被低估

6. 注意口径问题/数据及时利用问题。口径问题是指：在财报发布的月份中，比如4月，A公司比B公司先发，此时B公司还是t-1期的财务数据，而A公司是t期的财务数据，由于Brain是日频调仓，可能不具备可比性；但如果不及时利用数据，收益也会被其他交易者兑现，因此需要权衡。但大多数情况下，口径问题<<<<<<<<数据及时利用问题，因此该问题建议注意而不是规避

7. 在计量经济学中，还有“互为因果”的问题，我对这个问题的了解没有那么深入，不过直觉上很好理解。我们拿到的财务数据，是静态的，也就是说，我们不清楚这样的财务数据是如何形成的。例如成本越高，收益越高，互为因果现象会是：成本越高，我的收益理应变高，收益变高，我就会更激进地投入生产，提高成本，这也会导致估计量有误。传统的解决方法为引入额外的工具变量，在做alpha中，我们也许可以融合另类数据，比如财务附注中的语气，判断公司是激进扩张还是稳扎稳打，从而决定 变量在回归中的位置
