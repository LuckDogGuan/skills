## 帖子正文

### 为帮助所有用户以及已加入的顾问正确理解 Consultant 这一兼职的定位、要求与边界，减少误解，统一行为预期，现就相关事项说明如下。本文既是建议分享，也是行为指引，希望所有相关人员认真阅读并据此规范自身行为。

### 任何违反平台指引或侵犯平台知识产权等行为视情节严重程度，可能会导致风控面试、账户冻结、顾问合同中止等结果。盗用他人身份等违法行为可能带来进一步的后果。

首先需要明确， **Consultant 是一项兼职工作，与投资本身没有直接关系** 。它不是投资课程，不是投资培训，也不会直接帮助你的个人投资表现。若加入这一兼职的主要目的是“学习投资”“获取投资建议”或“借此提升个人投资收益”，那么这里并不是合适的地方。Consultant 的核心，是围绕平台资源（数据、回测框架等）开展研究、提交成果，并不适合个人投资者。

同时， **对于收入也应保持长期和合理预期** 。这不是一个赚快钱的地方。从申请、熟悉平台、完成培训、开始积累 value factor，到有机会获得季度奖励，通常需要数月时间，很多人拿到第一笔收入往往也发生在几个月之后。无论是能力、知识，还是决定收入表现的 value factor 与 weight，都需要长期积累，不可能在短时间内自然形成。因此，Consultant 更适合具备长期主义精神、愿意持续学习和迭代的人，而不适合期待短期变现、快速回报的人。

从适配性来看，这项兼职更适合 **愿意学习新东西、能够依靠自我驱动持续投入的人** 。这里不是学校，也不是传统公司。没有人会持续布置任务，也没有人按固定节奏考核你的 KPI。平台会提供资源和路径，但不会替代你完成学习、思考和积累。很多人反复询问“论坛要不要看”“零基础要不要学”“tutorial 要不要看”，这些问题本身其实反映出对这项工作的本质理解还不够清楚。所有学习资源都是你的资源，如何使用、能否吸收、能否转化为能力，取决于你自己。你是在和平台上的其他顾问共同竞争，最终资源和收入会更自然地流向那些能够提交更高质量 alpha、创造更高价值的人。

对于顾问来说， **真正应该做的，是沿着官方培训路径完成基础建设，并在此基础上形成自己的方法和工作体系** 。对于零基础用户，应依次重视并学习“零基础学量化”“新顾问衔接课”“AI 打工人”“BRAIN API 精讲和三阶”“从数据出发寻找 alpha”等内容。官方培训的作用，是帮助大家建立基础认知、熟悉平台、理解方法、完成入门。但培训只是起点，不是终点。真正能够长期做好的 Consultant，都不是靠简单照搬课程，而是在实践中不断尝试、不断总结、不断复盘，最终形成适合自己的理解、方法和流程。

与此同时，顾问也应当 **主动尝试、主动积累、主动复盘** 。平台培训只能带你入门，真正的成长来自持续实践。多和其他人交流、多看别人分享的经验是有帮助的，但交流的目的不是复制，而是吸收之后形成自己的判断。别人的经验可以借鉴，但不能代替自己的研究。只有真正形成独立理解和实际研究能力，才能具备长期竞争力。对于 VF 的波动也应理性看待。VF 有高有低是正常现象，阶段性的起伏本身就是复盘和优化的机会。遇到低谷时，重点不是放弃，而是回到过程本身，检查自己的思路、方法和执行环节，持续改进。

同样重要的是， **必须明确哪些事情不应该做，哪些行为平台会严厉打击** 。

第一， **不应走捷径、耍小聪明** 。例如通过向别人索要 alpha、私下交换 alpha、以凑分和凑提交为目的进行形式化操作，而没有自己的真实学习和研究。这类行为表面上看似提高了提交数量，实质上并不会建立真正能力，也背离了 Consultant 机制设立的初衷。

第二， **严禁开小号及相关规避行为** 。每个人都只能有一个账户。重复开户、冒用他人身份开户、使用他人信息注册小号，都是明确违反规则的行为，一经发现将被严厉处理。平台不会容忍任何通过账户规避规则、扰乱公平性的行为。

第三， **严禁非法牟利，以及任何损害社区和平台知识产权的行为** 。包括但不限于未经允许传播、出售、转让、盗用平台相关内容、研究成果或其他受保护资源。这不仅损害社区秩序，也直接破坏平台运行的基本规则。

第四， **成为顾问后，必须对自己提交的 alpha 负责** 。用户阶段的 alpha 本质上是练手和熟悉机制的过程，属于“新手村”阶段；这一阶段的 alpha 不会被采用，也没有机会进入生产和实盘。因此，用户阶段的尝试本身允许存在学习性质和试验性质。但成为顾问后，如果长期提交低质量 alpha、对研究缺乏基本理解、对自己提交的alpha不负责任，就不再只是个人学习问题，而会对平台整体质量造成损害。

第五， **不应把工作理解为单纯刷脚本** 。如果只会机械地刷脚本，对 alpha research 本身几乎一无所知，不理解自己在提交什么，也不理解结果背后的逻辑，那么这种行为不符合顾问角色应有的要求。平台需要的是对研究有基本认知、能对提交负责、能持续提升质量的顾问。

第六， **不应长期挂机，或表现出与实际提交行为明显不匹配的异常状态** 。如果长期缺乏真实投入，却持续以不合理方式维持表面活跃，或者存在其他与正常研究行为不匹配的情况，不仅是在浪费平台算力资源，同样会影响平台对其行为真实性、合规性和质量的判断。

总的来说，我们希望大家理解： **Consultant 不是投资通道，不是赚快钱的机会，也不是靠投机取巧就能长期获益的岗位** 。它的本质，是一个需要长期学习、持续积累、独立思考和对结果负责的兼职角色。平台欢迎认真学习、踏实研究、长期投入的人；对于破坏公平、规避规则、损害平台和社区环境的行为，也会持续保持明确、严格的打击态度。

希望所有用户和顾问都能建立正确认知，在清晰边界内开展自己的学习和工作，共同维护一个专业、健康、有长期价值的社区环境。
