### 评论 #2 (作者: UG81605, 时间: 7个月前)

I have been through this. I am a user from more than 3 years. Have been used websim, the og platform too.  The alpha that you submit takes 6 months minimum to accrue weights. Its not necessary that each alpha will get some weight. It has to pass all their criteria, stringent OS tests and then it will accrue some weight. Plus OS tests are of 2 years if im not wrong. So weight will refresh daily but for to start from zero it takes some time. 
So better is to focus on value factor, that is the key to higher payments and that is what is in your hands to change.
