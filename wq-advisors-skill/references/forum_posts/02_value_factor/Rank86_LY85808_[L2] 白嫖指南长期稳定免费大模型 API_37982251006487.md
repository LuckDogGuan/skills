## 国内 API 厂商/平台 (推荐)

速度快，中文支持好，主力推荐。

### S 级推荐

- **阿里心流 Iflow [https://platform.iflow.cn/models](https://platform.iflow.cn/models)**
  - **评价** : 目前已知免费额度最大、速度极快的平台。
  - **主力模型** : 千问全系列, Kimi-K2-Instruct-0905, GLM-4.6, DeepSeek-V3.2-Exp。
  - **:fire: Iflow-CLI** : 支持最新的 glm-4.7 / minimax-m2.1 (可配合 CLIProxyAPI 转发使用，非常稳)。
  - **限制** : 单并发 (同时只能发起一个请求)。

### 主力备选

- **字节火山引擎 (方舟) [https://www.volcengine.com/docs/82379/1399514?lang=zh](https://www.volcengine.com/docs/82379/1399514?lang=zh)**
  - **额度** : 单模型每日  **250w token** 。
  - **模型** : 豆包系列, deepseek-v3.2, Kimi-K2。
  - **缺点** : 单模型 token 耗尽需切换模型，略麻烦。
- **阿里 ModelScope (魔搭)**
  - **额度** : 每日总计  **2000**  次调用 (单模型限 500 次，部分大模型限 200 次)。
  - **评价** : 额度极其稳定，适合保底。
- **七牛云 AI**  (:hot_springs: 特别提及)
  - **福利** : 官方提供  **300w 免费 token**  (有效期一年)。
  - **亮点** : 实测支持  **OpenAI/Claude/Gemini**  模型 (需在控制台资源包查看适用范围)，速度可达 100+ token/s。

## 国外 API 厂商/平台

需解决网络问题，部分模型独占。

### :trophy: 推荐关注

- **NVIDIA NIM API [https://build.nvidia.com/models](https://build.nvidia.com/models)**
  - **评价** : 显卡老黄的福利，比 OpenRouter 更“香”。
  - **模型** : glm-4.7, minimax-m2.1, deepseek-v3.2, mistral-large 等。
  - **文生图** : FLUX.1-dev (免费 25 requests)。
- **Cerebras Inference**
  - **评价** :  **速度之王**  (220+ token/s)。
  - **模型** : glm-4.6, qwen-3, gpt-oss-120b。
  - **限制** : 每日 100w token (刷起来有点不够用)。

### 主流聚合/厂商

- **OpenRouter** : 行业标杆。未充值用户每日  **50 rpd**  (仅限名字带 free 的模型)。
- **Mistral** : 欧洲大厂。
  - **槽点** : 网页版 Le Chat 体验一般，模型效果个人感觉不如国内 Qwen/DeepSeek。
  - **额度** : API 免费额度惊人 (约每日 3300w token)，适合特定需求。
- **Groq** : 推理速度快，但免费的是小模型居多，且每日 token 限制较严 (100k-500k)。
- **Poe** : 每日免费 3000 积分。
  - **Tips** : 推荐选 Grok-4.1-Fast, Gemini Flash, GPT-5-mini 等低耗分模型。
  - **:bug: 警告** : 不支持结构化输出 (JSON Schema)，开发慎用。

### 观察区 (不太稳定)

- **ZenMux** : 提供 Gemini/Xiaomi/Kuaishou 的 Flash 版模型，偶发 429 错误。
- **Chutes** : 免费规则变动频繁，目前已取消所有免费模型 (2025-12-30 更新)。

## 附录：常见限制说明

- **RPM (Requests per minute)** : 每分钟请求次数
- **RPD (Requests per day)** : 每天请求次数
- **TPM (Tokens per minute)** : 每分钟 Token 数
- **TPD (Tokens per day)** : 每天 Token 数
